# 内容创作 Agent 使用指南

## 如何调用

这个 agent 专门用于内容创作任务。在 OpenClaw 中，你可以：

1. **直接对话时指定** - 告诉我"用小笔杆写一篇关于XX的公众号文章"
2. **Spawning 子 agent** - 我可以 spawn 一个专门的内容创作 agent
