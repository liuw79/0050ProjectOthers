## 教训与经验

### 2026-03-01 小红书运营
1. **永远不要提"伟哥手动发布"这样的方案** - 伟哥请我来是给他干活的，不是让他给我打工。单靠自己，不要指望他。
2. **发布后必须验证** - 不要只看"发布成功"提示，必须去笔记管理页面确认笔记真的在列表中。
3. **标题字数要检查** - 小红书标题有字数限制，发布前要确认。
4. **绝对不要使用Ctrl+A或Command+A全选** - 无论什么情况下，全选操作都会导致输入多余字符"a"，破坏标题或正文。应该直接点击输入框，然后直接输入新内容。
5. **封面描述只输入核心文字** - "文字配图"功能会把输入的文字直接显示在封面上，不要加"小红书封面："、"简洁商务风格"这样的描述词，只输入想要显示在封面上的核心文字。
6. **账号切换要谨慎** - 确保发布到正确的账号（deepseek），不要发布到错误账号（冯卫东讲定位）。
