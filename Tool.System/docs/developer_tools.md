# 开发者工具

### 🐙 GitHub Desktop

**功能**：Git 图形界面客户端

**特点**：
- 简单易用
- GitHub 集成
- 分支管理
- 冲突解决

**下载地址**：[官网](https://desktop.github.com/)

### 🗄️ HeidiSQL

**功能**：数据库管理工具

**特点**：
- 多数据库支持
- SQL 编辑器
- 数据导入导出
- 免费开源

**下载地址**：[官网](https://www.heidisql.com/)

### 🔧 Windows Terminal

**功能**：现代终端应用

**特点**：
- 多标签页
- 自定义主题
- Unicode 支持
- GPU 加速

**下载地址**：[Microsoft Store](https://aka.ms/terminal)
