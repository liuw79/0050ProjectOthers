# 系统增强工具

## 桌面日历背景设置

### 方法一：WallCal（推荐）
**特点：** 专业的桌面日历应用，半透明覆盖在壁纸上
- **下载：** Mac App Store搜索"WallCal"
- **价格：** 免费基础版，专业版需付费
- **功能：**
  - 半透明背景日历覆盖在桌面壁纸上
  - 显示当前macOS/iCloud日历事件和提醒
  - 支持键盘/鼠标切换前台/后台显示
  - 可调整大小和位置
  - 专业版支持周视图、自定义颜色、选择性显示日历
- **安装命令：** `open "https://apps.apple.com/cn/app/wallcal-wallpaper-calendar/id1581367286"`

### 方法二：Plash + 网页日历（免费）
**特点：** 将任何网页设为桌面壁纸，包括日历
- **下载：** GitHub或Mac App Store搜索"Plash"
- **价格：** 免费开源
- **使用步骤：**
  1. 安装Plash应用
  2. 打开Google Calendar或其他网页日历
  3. 在Plash中添加日历网页URL
  4. 设置为桌面壁纸
- **优势：** 可显示Google Calendar、Outlook 365等任何网页日历
- **GitHub：** https://github.com/sindresorhus/Plash

### 方法三：DejaDesktop
**特点：** 专业的桌面信息显示工具
- **功能：**
  - 支持iCal、Google、Outlook日历
  - 可显示月视图、日视图
  - 自动更新，无需手动刷新
  - 支持多个日历源同时显示
- **网站：** https://www.dejadesktop.com/mac.html

### 安装建议
1. **简单需求：** 选择WallCal，界面美观，功能完整
2. **免费方案：** 选择Plash + Google Calendar
3. **专业用户：** 选择DejaDesktop，功能最全面

### 注意事项
- 所有方案都支持macOS 11+
- WallCal和DejaDesktop不支持编辑日历，仅显示
- Plash方案需要网络连接才能显示在线日历
- 建议先试用免费版本，确认满足需求后再购买付费功能
