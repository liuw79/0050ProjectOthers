# 快速启动工具

### 🚀 PowerToys Run

**功能**：快速启动器和搜索工具

**特点**：
- 微软官方工具
- 支持应用程序、文件、计算器等
- 插件系统
- 快捷键：`Alt+Space`

**下载地址**：[Microsoft Store](https://aka.ms/powertoys) 或 [GitHub](https://github.com/microsoft/PowerToys)

### 🔍 Listary

**功能**：文件搜索和快速访问工具

**特点**：
- 实时文件搜索
- 快速文件夹导航
- 支持关键词搜索
- 双击 `Ctrl` 激活

**下载地址**：[官网](https://www.listary.com/)

### ⚡ Wox

**功能**：开源快速启动器

**特点**：
- 完全免费
- 插件丰富
- 支持网页搜索
- 快捷键：`Alt+Space`

**下载地址**：[GitHub](https://github.com/Wox-launcher/Wox)
