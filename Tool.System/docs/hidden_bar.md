# Hidden Bar 使用指南

## 什么是那个"小于号"图标？

您在菜单栏看到的 **"<"** 图标就是 Hidden Bar 的**分隔符**，它的作用是：
- 将菜单栏分为两个区域：**可见区域**（左侧）和**隐藏区域**（右侧）
- 点击它可以显示/隐藏右侧的图标

## 如何使用 Hidden Bar 隐藏 Monica 图标

### 步骤 1：找到分隔符
在菜单栏右上角找到 **"<"** 图标（这就是 Hidden Bar 的分隔符）

### 步骤 2：拖拽 Monica 图标
1. **按住** Monica 图标
2. **拖拽** Monica 图标到分隔符 **"<"** 的**左侧**
3. 松开鼠标

### 步骤 3：隐藏图标
- **点击** 分隔符 **"<"**
- 分隔符左侧的图标就会被隐藏起来

### 步骤 4：显示图标（需要时）
- 再次**点击** 分隔符，隐藏的图标就会重新显示

## 图解说明

```
隐藏前的菜单栏：
[其他图标] [Monica图标] [<] [系统图标]
           ↑需要隐藏    ↑分隔符

拖拽后的菜单栏：
[Monica图标] [其他图标] [<] [系统图标]
↑已移到左侧            ↑分隔符

点击分隔符后：
[其他图标] [<] [系统图标]
          ↑分隔符（Monica图标已隐藏）

再次点击分隔符：
[Monica图标] [其他图标] [<] [系统图标]
↑重新显示             ↑分隔符
```

## 重要提示

1. **拖拽方向**：一定要将图标拖到分隔符的**左侧**
2. **点击控制**：点击分隔符来控制左侧图标的显示/隐藏
3. **永久设置**：Hidden Bar 会记住您的设置，重启后依然有效

## 如果遇到问题

### 问题1：找不到分隔符
- 确保 Hidden Bar 正在运行
- 检查菜单栏右侧是否有 "<" 图标

### 问题2：无法拖拽图标
- 某些系统图标可能无法移动
- 尝试按住 Command 键再拖拽

### 问题3：图标没有隐藏
- 确保图标已拖拽到分隔符右侧
- 点击分隔符来隐藏右侧图标

## 恢复原始状态

如果想要卸载 Hidden Bar：
```bash
# 退出 Hidden Bar
pkill "Hidden Bar"

# 卸载应用
brew uninstall --cask hiddenbar
```

## 总结

Hidden Bar 的核心概念：
- **"<" 图标** = 分隔符
- **左侧** = 可以隐藏的图标
- **右侧** = 始终可见的图标
- **点击分隔符** = 切换左侧图标的显示状态
