# Chrome书签重复清理工具

## 📋 项目概述

这是一个专门用于清理Chrome浏览器重复书签的工具集，能够自动检测、分析和移除重复的书签，帮助您保持书签的整洁和有序。

## 🎯 功能特点

- ✅ **智能检测**: 基于URL自动检测重复书签
- ✅ **安全备份**: 操作前自动创建备份文件
- ✅ **保留策略**: 自动保留最新添加的书签
- ✅ **详细报告**: 生成完整的分析和清理报告
- ✅ **非破坏性**: 生成清理后文件，不直接修改原文件

## 🛠️ 工具文件说明

### 1. `analyze_bookmarks.py`
**功能**: 仅分析重复书签，不进行修改
```bash
python3 analyze_bookmarks.py
```
- 扫描所有书签
- 检测重复项
- 生成详细报告
- 不修改任何文件

### 2. `auto_clean_bookmarks.py`
**功能**: 自动清理重复书签
```bash
python3 auto_clean_bookmarks.py
```
- 自动备份原文件
- 移除重复书签
- 保留最新版本
- 生成清理后文件

### 3. `chrome_bookmark_cleaner.py`
**功能**: 交互式清理工具
```bash
python3 chrome_bookmark_cleaner.py
```
- 提供多种操作模式
- 支持不同保留策略
- 可选择是否应用到Chrome

## 🔧 技术细节

### 检测算法
- **URL标准化**: 将URL转换为小写进行比较
- **重复判定**: 相同URL被视为重复
- **保留策略**: 默认保留添加时间最新的书签

### 文件格式
- Chrome书签使用JSON格式存储
- 支持嵌套文件夹结构
- 保持原有的文件夹组织

### 安全措施
- 操作前自动备份
- 非破坏性处理
- 详细的操作日志
