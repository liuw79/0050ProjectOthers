# 四指手势使用说明

## 功能描述
四指下滑关闭当前活动窗口功能，类似于移动设备上的手势操作。

## 使用方法
1. 在触控板上用四个手指向下滑动
2. 当滑动距离超过阈值时，会自动关闭当前活动窗口
3. 相当于按下 `Cmd + W` 快捷键

## 技术实现
- 使用 Hammerspoon 原生事件监听器
- 监听四指触摸手势事件
- 计算滑动距离和方向
- 触发窗口关闭操作

## 配置参数
- **手指数量**: 4个手指
- **滑动方向**: 向下
- **距离阈值**: 0.15（可在代码中调整）
- **触发动作**: `Cmd + W`

## 故障排除

### 1. 手势无响应
**可能原因**:
- Hammerspoon 没有辅助功能权限
- 系统四指手势被其他应用占用
- 触控板设置问题

**解决方法**:
1. 打开 `系统偏好设置` > `安全性与隐私` > `隐私` > `辅助功能`
2. 确保 Hammerspoon 已添加并勾选
3. 检查 `系统偏好设置` > `触控板` > `更多手势` 中的设置

### 2. 手势过于敏感或不够敏感
**解决方法**:
在 `init.lua` 文件中调整 `threshold` 值：
- 增大数值：降低敏感度，需要更大的滑动距离
- 减小数值：提高敏感度，较小的滑动即可触发

```lua
local gestureState = {
    startY = nil,
    isTracking = false,
    threshold = 0.15 -- 调整这个值
}
```

### 3. 与系统手势冲突
**解决方法**:
1. 打开 `系统偏好设置` > `触控板` > `更多手势`
2. 关闭或修改 `App Exposé` 设置
3. 或者在代码中修改手势检测逻辑

### 4. 调试信息
手势触发时会在 Hammerspoon 控制台显示调试信息：
- `🖐️ 四指手势开始，起始Y: xxx`
- `⬇️ 检测到四指下滑，距离: xxx`
- `✅ 关闭窗口: 窗口标题`

查看控制台：点击菜单栏 Hammerspoon 图标 > `Console`

## 自定义配置

### 修改触发手指数量
```lua
if not touches or #touches ~= 4 then -- 改为其他数字
```

### 修改触发方向
```lua
local deltaY = gestureState.startY - avgY -- 下滑为正值
-- 改为上滑检测：local deltaY = avgY - gestureState.startY
```

### 修改触发动作
```lua
hs.eventtap.keyStroke({"cmd"}, "w") -- 改为其他快捷键
```

## 注意事项
1. 需要 macOS 系统支持多点触控
2. 需要授予 Hammerspoon 辅助功能权限
3. 可能与系统默认手势冲突，需要适当调整
4. 仅对标准窗口有效，某些特殊窗口可能无法关闭
