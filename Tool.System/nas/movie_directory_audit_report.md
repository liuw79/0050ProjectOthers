# 电影目录全面排查报告

## 排查概述

本次对 `/Volumes/video/电影/` 目录进行了全面排查，发现了多个需要整理的问题。电影目录规模庞大，结构复杂，需要制定详细的整理计划。

## 基本统计信息

- **总目录数量**: 476个一级目录
- **多级子目录**: 258个二级和三级子目录
- **技术规格目录**: 65个包含技术信息的目录名
- **空目录**: 2个空目录需要清理

## 发现的主要问题

### 1. 命名规范不统一

**问题描述:**
电影目录存在多种命名格式，缺乏统一标准：

- **中文简洁格式**: `阳光灿烂的日子 (1994)`
- **技术规格格式**: `战狼.Wolf.Warriors.2015.WEB-DL.2160p.H265.10bits.HDR.DDP 5.1-PTHweb (2015)`
- **英文格式**: `12 Angry Men (1957)`
- **混合格式**: `唐人街探案3.Detective.Chinatown.3.2021.WEB-DL.2160p.H265.10bit.HDR10.DDP Atmos 5.1-PTHweb (2021)`
- **特殊标记**: `[热辣滚烫].yoLo.2024.2160p.WEB-DL.HEVC.10bit.DV.DDP5.1.Atmos.2Audios-QHstudIo`

### 2. 系列电影分散存储

**发现的系列电影:**
- **战狼系列**: 战狼1 (2015) 和 战狼2 (2017) 分别存储
- **这个男人来自地球系列**: 原版 (2007) 和续集 (2017) 分别存储
- **Star Trek系列**: 存储在独立的 `star trek` 子目录中（包含10部电影）

### 3. 特殊目录结构

**合集目录:**
- `[合集] TLF 周星驰系列` - 包含周星驰电影合集
- `IMDB.TOP250.2020.09 PART.I (001-125)` - IMDB排行榜合集（但为空目录）

**技术分类目录:**
- `m2ts` - 按文件格式分类
- `1080.AVC` - 按技术规格分类
- `1080p (2016)` - 混合年份和技术规格

**临时/无用目录:**
- `新建文件夹` - 临时目录
- `.deletedByTMM` - 媒体管理软件删除目录
- `@lavender27 (2021)` - 用户标记目录

### 4. 年份分布分析

**热门年份统计:**
- 2021年: 30部电影
- 2020年: 20部电影
- 2019年: 20部电影
- 2018年: 19部电影
- 2015年: 16部电影
- 1994年: 16部电影

### 5. 空目录问题

**需要清理的空目录:**

- `Someone.To.Talk.To.2016.4K&1080p.HEVC&AVC.4in1.WEB-DL.x264.AAC-HQC`

## 整理计划

### 阶段一：清理和预处理

1. **清理空目录**
   - 删除确认为空的目录
   - 检查是否有隐藏文件

2. **清理临时目录**
   - 删除 `新建文件夹`
   - 处理 `.deletedByTMM` 目录
   - 清理用户标记目录

3. **技术规格目录处理**
   - 评估 `m2ts`、`1080.AVC` 等技术分类目录的必要性
   - 决定是否合并到主目录

### 阶段二：命名标准化

1. **制定统一命名规范**
   ```
   标准格式: 电影名称 (年份)
   示例: 阿甘正传 (1994)
   英文电影: Forrest Gump (1994)
   ```

2. **技术信息处理策略**
   - 保留重要的质量信息（4K、HDR）
   - 简化过长的技术规格
   - 统一技术标记格式

3. **特殊字符处理**
   - 移除方括号 `[]`
   - 统一点号和空格使用
   - 处理特殊符号

### 阶段三：系列电影整理

1. **创建系列目录**
   ```
   战狼系列/
   ├── 战狼 (2015)/
   └── 战狼2 (2017)/
   
   这个男人来自地球系列/
   ├── 这个男人来自地球 (2007)/
   └── 这个男人来自地球：全新纪 (2017)/
   ```

2. **保持现有合集**
   - 保留 `Star Trek` 系列的独立目录结构
   - 优化周星驰合集的组织方式

### 阶段四：分类优化

1. **按类型分类（可选）**
   - 考虑是否需要按类型创建子目录
   - 评估按年代分类的可行性

2. **质量分级（可选）**
   - 4K/HDR 高质量电影
   - 1080p 标准质量电影
   - 其他格式电影

## 风险评估

### 高风险操作
- 大量重命名操作可能影响媒体库索引
- 移动文件可能破坏现有的播放列表
- 删除操作不可逆

### 建议预防措施
- 操作前完整备份目录结构
- 分批次执行，每次处理少量文件
- 保留操作日志，便于回滚
- 测试媒体播放器的兼容性

## 实施建议

### 优先级排序
1. **高优先级**: 清理空目录和临时文件
2. **中优先级**: 系列电影整理
3. **低优先级**: 命名标准化
4. **可选**: 分类重组

### 工具开发需求
1. **电影目录分析器** - 分析命名模式和重复内容
2. **智能重命名工具** - 基于规则的批量重命名
3. **系列检测器** - 自动识别系列电影
4. **安全操作工具** - 带回滚功能的文件操作

## 预期效果

整理完成后，电影目录将具备：
- 统一的命名规范
- 清晰的系列电影组织
- 更好的浏览和搜索体验
- 减少重复和冗余内容
- 提高媒体库管理效率
