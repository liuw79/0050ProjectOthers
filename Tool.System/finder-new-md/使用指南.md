# Finder 右键新建 Markdown 文件 - 快速指南

## ✅ 安装状态

已成功安装！快速操作已添加到系统服务中。

## 🚀 立即使用

### 方式一：右键菜单（推荐）

1. 打开 Finder，进入任意文件夹
2. 在空白处右键点击
3. 选择「快速操作」→「新建 Markdown 文件」
4. 文件会自动创建并进入重命名状态

![右键菜单](https://via.placeholder.com/600x400/f0f0f0/333333?text=Finder+右键+→+快速操作+→+新建+Markdown+文件)

### 方式二：选中文件夹后右键

1. 在 Finder 中选中一个文件夹
2. 右键点击该文件夹
3. 选择「快速操作」→「新建 Markdown 文件」
4. 文件会在该文件夹内创建

### 方式三：设置快捷键（可选）

让操作更快捷：

1. 打开「系统设置」（或「系统偏好设置」）
2. 点击「键盘」
3. 左侧选择「键盘快捷键」→「服务」
4. 在右侧找到「新建 Markdown 文件」
5. 双击右侧，按下你想要的快捷键组合（例如：⌘⇧M）

推荐快捷键：
- `⌘⇧M` - Markdown 的首字母
- `⌘⌃N` - New 的首字母
- `⌘⌥M` - 其他组合

## 📝 文件模板

创建的 Markdown 文件包含以下默认模板：

```markdown
# 标题

## 简介

内容...

## 详细信息

```

### 自定义模板

如果想修改默认模板，编辑以下文件：

```bash

open -e ~/SynologyDrive/0050Project/Tool.System/finder-new-md/create_new_md.sh
```

找到这部分并修改：

```bash

```

## ❓ 常见问题

### Q: 为什么看不到「新建 Markdown 文件」选项？

**A:** 尝试以下方法：

1. **等待片刻**：系统需要几秒钟来识别新服务
2. **重启 Finder**：
   ```bash
   killall Finder
   ```
3. **注销并重新登录**：最可靠的方法
4. **检查安装**：
   ```bash
   ls -la ~/Library/Services/ | grep Markdown
   ```

### Q: 可以修改默认文件名吗？

**A:** 可以！编辑 `create_new_md.sh`：

```bash
base_name="未命名"  # 改为 "新文档" 或其他名称
```

### Q: 如何在多个位置使用？

**A:** 此功能在 Finder 的所有位置都可用：
- 本地文件夹
- 网络共享
- iCloud Drive
- 外部硬盘
- 任何 Finder 可访问的位置

### Q: 文件名冲突怎么办？

**A:** 自动处理！如果文件已存在，会添加数字后缀：
- `未命名.md`
- `未命名1.md`
- `未命名2.md`
- ...

### Q: 可以创建其他类型的文件吗？

**A:** 可以！复制整个 `finder-new-md` 文件夹，修改：
- 文件扩展名（`.txt`, `.py`, `.js` 等）
- 文件模板内容
- 快速操作名称
