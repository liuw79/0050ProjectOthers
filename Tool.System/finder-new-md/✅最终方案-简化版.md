## 🚀 使用方法

### 方式 1：Finder 菜单（立即可用）

1. 在 Finder 中
2. 点击菜单栏「Finder」→「服务」
3. 选择「新建 md (新建 md.workflow)」
4. ✅ 文件创建成功！

### 方式 2：设置快捷键（可选）

如果需要快捷键：

1. 打开「系统设置」→「键盘」→「键盘快捷键」
2. 左侧选择「服务」
3. **展开「文件和文件夹」分类**
4. 找到「新建 md」并设置快捷键
