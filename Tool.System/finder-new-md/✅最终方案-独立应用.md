## 🚀 使用方法

### 方式 1：拖到 Dock（最推荐）⭐⭐⭐⭐⭐

1. 打开 `~/Applications/` 文件夹
2. 找到「新建MD文件.app」
3. **拖到 Dock 栏**
4. 需要时点击即可创建 MD 文件

### 方式 2：双击运行

1. 在 Finder 中打开 `~/Applications/`
2. 双击「新建MD文件.app」
3. 会在当前 Finder 目录创建 MD 文件

### 方式 3：拖到 Finder 工具栏

1. 打开 Finder
2. 按住 `⌘` 键
3. 将「新建MD文件.app」拖到 Finder 工具栏
4. 以后点击工具栏按钮即可
