# 右键菜单没有「快速操作」- 解决方案

## 问题描述
macOS 26.2 (你的版本) 中，Finder 右键菜单可能不显示「快速操作」选项。

---

## 解决方案汇总

### 🎯 方案 1：通过「服务」菜单访问（立即可用）

即使右键菜单中没有「快速操作」，服务依然可以通过其他方式访问：

#### 方式 A：Finder 菜单栏
1. 在 Finder 中，点击顶部菜单栏的 **「Finder」**
2. 选择 **「服务」**（或「Services」）
3. 应该能看到「新建md」
4. 点击即可创建文件

#### 方式 B：使用快捷键
1. 打开「系统设置」→「键盘」→「键盘快捷键」
2. 左侧选择「服务」
3. 找到「新建md」
4. 双击右侧，设置快捷键（如 `⌘⇧M`）
5. 以后在 Finder 中按快捷键即可

---

### 🎯 方案 2：启用右键菜单中的「服务」

#### 步骤：

1. **打开系统设置**
   - 点击  菜单 → 系统设置

2. **进入键盘设置**
   - 点击「键盘」

3. **配置键盘快捷键**
   - 点击「键盘快捷键」按钮

4. **检查服务设置**
   - 左侧选择「服务」
   - 在右侧找到「新建md」
   - **确保勾选框是打钩的**

5. **重启 Finder**
   ```bash
   killall Finder
   ```

---

### 🎯 方案 3：修改 Finder 右键菜单设置

某些 macOS 版本需要手动启用服务显示：

```bash
# 启用 Finder 中的服务菜单
defaults write com.apple.finder ShowServicesInContextMenu -bool true

# 重启 Finder
killall Finder
```

运行后，右键菜单应该会出现「服务」或「快速操作」选项。

如果你使用 Alfred 或 Raycast：

#### Alfred：

2. 运行脚本：
   ```bash
   /Users/comdir/SynologyDrive/0050Project/Tool.System/finder-new-md/create_new_md.sh "$(osascript -e 'tell application "Finder" to POSIX path of (insertion location as alias)')"
   ```

1. 创建自定义脚本命令
2. 使用相同的脚本路径

### 🎯 方案 5：创建 Finder 工具栏按钮

这个方法非常直观！

#### 步骤：

1. **将脚本转为应用**
   ```bash
   cd /Users/comdir/SynologyDrive/0050Project/Tool.System/finder-new-md
   
   # 创建应用包
   mkdir -p "新建MD.app/Contents/MacOS"
   
   # 复制脚本
   cp create_new_md.sh "新建MD.app/Contents/MacOS/新建MD"
   chmod +x "新建MD.app/Contents/MacOS/新建MD"
   
   # 创建 Info.plist
   cat > "新建MD.app/Contents/Info.plist" << 'EOF'
   <?xml version="1.0" encoding="UTF-8"?>
   <!DOCTYPE plist PUBLIC "-//Apple//DTD PLIST 1.0//EN" "http://www.apple.com/DTDs/PropertyList-1.0.dtd">
   <plist version="1.0">
   <dict>
       <key>CFBundleExecutable</key>
       <string>新建MD</string>
       <key>CFBundleName</key>
       <string>新建MD</string>
       <key>CFBundlePackageType</key>
       <string>APPL</string>
   </dict>
   </plist>
   EOF
   ```

2. **拖到 Finder 工具栏**
   - 按住 `⌘` 键
   - 拖动「新建MD.app」到 Finder 工具栏
   - 以后点击工具栏按钮即可

## 🚀 推荐操作顺序

### 第一步：立即尝试（无需配置）
1. 打开 Finder
2. 点击菜单栏「Finder」→「服务」
3. 查看是否有「新建md」

### 第二步：启用右键菜单（1分钟）
```bash
defaults write com.apple.finder ShowServicesInContextMenu -bool true
killall Finder
```

### 第三步：设置快捷键（2分钟）
1. 系统设置 → 键盘 → 键盘快捷键 → 服务
2. 找到「新建md」
3. 设置为 `⌘⇧M`

### 第四步：测试
在 Finder 中按 `⌘⇧M`，应该会创建 MD 文件！
