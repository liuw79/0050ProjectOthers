# 🎯 Finder 新建 Markdown 文件 - 最终使用指南

## ✅ 安装状态

- ✅ 服务已创建：`~/Library/Services/新建md.workflow`
- ✅ Finder 服务菜单已启用
- ✅ 系统服务已刷新

---

## 🚀 三种使用方式

### 方式 1：Finder 菜单栏（立即可用）✨

1. 打开 Finder，进入任意文件夹
2. 点击顶部菜单栏的 **「Finder」**
3. 选择 **「服务」**（Services）
4. 点击 **「新建md」**
5. ✅ 文件创建成功！

> 这是最可靠的方式，不需要任何额外配置。

---

### 方式 2：右键菜单

已启用 Finder 服务菜单显示，现在：

1. 在 Finder 中右键点击空白处
2. 查看菜单底部是否有 **「服务」** 或 **「快速操作」**
3. 选择 **「新建md」**

> 如果右键菜单中没有显示，请使用方式 1 或方式 3。

---

### 方式 3：快捷键（最推荐）⭐⭐⭐⭐⭐

#### 设置步骤：

1. **打开系统设置**
   - 点击  → 系统设置

2. **进入键盘设置**
   - 点击「键盘」
   - 点击「键盘快捷键...」按钮

3. **配置服务快捷键**
   - 左侧选择「服务」（Services）
   - 右侧找到「新建md」（可能在「文件和文件夹」或「通用」分类下）
   - 确保勾选框打钩 ✓
   - 双击右侧快捷键区域
   - 按下你想要的组合键

4. **推荐快捷键**
   - `⌘⇧M` (Command + Shift + M) - Markdown 首字母
   - `⌘⌃N` (Command + Control + N) - New 的首字母  
   - `⌘⌥M` (Command + Option + M) - 其他组合

5. **使用**
   - 在 Finder 中按下设置的快捷键
   - ✅ 立即创建文件！

> 💡 **提示**：也可以双击运行 `设置快捷键.command` 文件，会自动打开系统设置。

## ❓ 常见问题

### Q1: 右键菜单中没有「服务」或「快速操作」

**A:** 使用方式 1（Finder 菜单栏）或方式 3（快捷键）。

也可以尝试：
```bash
defaults write com.apple.finder ShowServicesInContextMenu -bool true
killall Finder
```

### Q2: 菜单中找不到「新建md」

**A:** 检查服务是否被禁用：
1. 系统设置 → 键盘 → 键盘快捷键 → 服务
2. 找到「新建md」
3. 确保勾选框打钩 ✓

### Q3: 想修改默认文件名

**A:** 编辑 `create_new_md.sh`：
```bash
base_name="未命名"  # 改为你想要的名称
```


**桌面快捷方式：**
