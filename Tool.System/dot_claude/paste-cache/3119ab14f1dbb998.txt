📝 使用方法
在另一台电脑上执行以下步骤：

步骤 1：创建 .ssh 目录

mkdir -p ~/.ssh && chmod 700 ~/.ssh
步骤 2：创建私钥文件

nano ~/.ssh/id_rsa
然后将下面的私钥内容粘贴进去，保存退出（Ctrl+O, Enter, Ctrl+X）

步骤 3：设置权限

chmod 600 ~/.ssh/id_rsa
步骤 4：测试连接

ssh -T git@github.com
步骤 5：克隆仓库

git clone git@github.com:liuw79/0050ProjectOthers.git
✓ 成功提示：如果看到 "Hi liuw79! You've successfully authenticated" 说明配置成功！
