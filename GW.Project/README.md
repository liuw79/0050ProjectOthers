# 项目计划总览

用于记录与跟踪以下三个项目：

- AI落地（`plans/AI落地.md`）
- 训战AI（`plans/训战AI.md`）
- 新课室（`plans/新课室.md`）
 - 后端流程IT化（`plans/后端流程IT化.md`）

## 使用建议

- 每次有进展、决策或卡点，及时在对应项目文档更新。
- 建议统一状态枚举：`计划中` / `进行中` / `已完成` / `阻塞`。
- 关键进度建议按日期记录，里程碑按阶段记录，卡点要给出应对措施和负责人。

## 字段说明

- 负责人：主责人；如有协同可在备注补充。
- 状态：当前推进状态；阻塞时务必写明原因与所需支持。
- 关联里程碑：该进度或任务关联的里程碑名称，便于回溯。
- 计划日期/实际日期：用于比较偏差并复盘。
