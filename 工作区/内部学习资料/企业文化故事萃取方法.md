# 企业文化故事萃取方法

本文档整理了国内外关于萃取企业文化小故事的方法、技能及AI提示词资源。

---

## 一、国内方法与技巧

### 1.1 写作四大要诀

| 要诀 | 说明 |
|------|------|
| **口子开掘要小** | 选题角度要精准 |
| **出场人物要少** | 聚焦核心人物 |
| **故事情节要曲折** | 增加故事性和吸引力 |
| **细节要感人** | 通过生动描写引起共鸣 |

### 1.2 故事特点

- 讲求"新、奇、巧"
- 寓意深刻、人物典型、动人撼心
- 以小见大，潜移默化传达企业文化
- 篇幅短小精干（200字至千八百字）

### 1.3 收集策略

- 设立专职或兼职文化情报员定期收集
- 按月度/季度定期汇总
- 收集贴近员工实际工作场景的真实案例

### 1.4 萃取步骤（"3"+"4"法）

**找到3个场景：**
- 先构建核心文化理念体系
- 再持续进行理念落地

**作对4件事：**
1. 编写文化故事
2. 收集文化故事
3. 传播文化故事
4. 践行故事精神

## 三、AI提示词模板

### 3.1 提取核心故事（中文版）

```
你是一位专家级叙事分析师。从这个访谈记录中提取反映公司价值观和文化的引人入胜的故事。对于每个故事，请识别：

1. 挑战或情境
2. 采取的行动
3. 结果
4. 体现的潜在价值观或原则

请以短小、引人入胜的叙事形式呈现（每篇200-300字）。
```

### 3.2 提取核心故事（英文版）

```
You are an expert narrative analyst. Extract compelling stories from this interview transcript that reflect the company's values and culture. For each story, identify:

1. The challenge or situation
2. The action taken
3. The outcome
4. The underlying value or principle demonstrated

Present as short, engaging narratives (200-300 words each).
```

### 3.3 主题分析（中文版）

```
分析这份员工证言，并识别：

- 与公司文化相关的前3-5个反复出现的主题
- 说明每个主题的具体例子
- 情感基调以及关键要点

请以包含引言的结构化报告格式呈现。
```

### 3.4 主题分析（英文版）

```
Analyze this employee testimonial and identify:
- Top 3-5 recurring themes related to company culture
- Specific examples that illustrate each theme
- Emotional tone and key takeaways

Format as a structured report with quotes.
```

### 3.5 价值洞察提取（英文版）

```
Act as a corporate culture researcher. Extract stories from this content that demonstrate the company's core values. For each story:

1. Identify the company value it illustrates
2. Summarize the story in a compelling way
3. Explain how the story reinforces the value
4. Rate the story's impact (1-10) as a culture narrative

Focus on authentic, relatable examples that would resonate with employees.
```

## 五、实用技能要求

### 5.1 文字驾驭能力

- 需要在驾驭文字上下工夫
- 通过生动的描写、细致的刻画写出感人的故事
- 对大量生活素材加以提炼和升华

### 5.2 创作特色

- 以实际案例为背景
- 以幽默风趣为特色，增强吸引力
- 让员工产生共鸣
