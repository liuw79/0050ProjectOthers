# 圈子机制详解（组织进化）

## 1. 为什么要启动圈子进化？ (Why)
*   **敏捷组织跃迁**：从2025年的“团队化”向2026年的“敏捷组织”转变，通过清晰边界和决策力实现自我管理。
*   **适配2.0业务模型**：企业服务业务走通后，需要资源围绕客户价值快速响应与闭环。
*   **德鲁克理念落地**：践行“知识工作者自我管理”，让管理真正导向外部成果。

## 2. 什么是圈子与社区？ (What)
*   **去中心化**：不再有传统的部门中心或前中后台概念，组织由“圈子”和“社区”组成。
*   **圈子 (Circles) = 价值作战单元**：
    *   核心逻辑：价值创造。
    *   必须回答：“我的客户是谁？”、“我为客户创造了什么价值？”。
    *   分类：直接面对外部客户的**经营圈**，支持内部客户的**服务/支持圈**。
*   **社区 (Communities) = 伙伴成长家园**：
    *   关注伙伴成长，通过勋章系统认证专业技能。
    *   让成长和贡献显性化、公平化。

## 3. 圈子如何运作？ (How)
*   **军长/接口人机制**：
    *   军长：负责对齐目标、调度资源、排列优先级。若未设军长，由选举产生接口人。
*   **角色职责**：
    *   鼓励跨领域发挥，**一人多角**。
    *   原则：每人**不超过3个角色**。
    *   每个角色有明确职责、职权及成果界定。
*   **高内聚低耦合**：
    *   绝大部分工作在圈子内部闭环，减少圈间协同拉扯。
    *   决策发生在离现场最近的地方。
*   **科学的三会制度**：
    *   **决策会**、**复盘会**、**例会**。
    *   由认证主持人引导，导向共识与行动。

## 4. 2026年圈子架构初始化（V1.0）

### A. 经营圈（直接面向客户）
*   **区域经营圈**（华东、华南、华北）：
    *   核心抓手：**会员级公开课运营**。
    *   关键配置：**业务BP**（伙伴成长/招募/后台支持）、**产品BP**（课程产品开发/敏捷调整）。
    *   变化：华东/华南配置项目经理独立开班统筹；华北重新起步探索独特性。
*   **KA经营圈**（重点深耕）：
    *   定位：深耕企业服务（ToB），不同于区域经营。
    *   目标：挖掘千万级/百万级订单，培养专职**企业陪跑顾问**。
    *   重点行业：跨境、科创（苏州逆上市企业）、餐饮、LMI领导力。

### B. 产品圈（按板块切分）
*   **结构**：打破原有大产品部，分为 **业务突破类**、**战略经营类**、**组织人才类** 三大板块。
*   **目的**：与经营圈协同作战，培养产品经理业务体感。
*   **协同**：通过产品BP与经营圈拉通。

### C. 支持/服务圈
*   **师资服务圈**：含BD功能，归口管理师资。
*   **产品规划与策略圈**：负责产品规划和经营策略，协同三大产品板块。
*   **小老师孵化圈**：
    *   孵化内部成员（如Miya、慧余）成为小老师。
    *   设立**辅导员**角色（预备军，如丁丁）。
    *   目标：自研核心学科，拥有知识版权，高频迭代。

### D. 战略/破圈任务（独立运作）
*   **内容破圈**：利用AI重塑内容生产，打造深度案例。
*   **渠道破圈**：主动拓展合作（如投资机构、园区），解决拉新问题。
*   **AI加速圈**：战略重点，技术+业务结合（FDE模式），不直接对外售卖能力，先改造内部产品。

### E. 其他
*   **效能圈**：基础建设，提升组织效能。
*   **PMO**：承担复杂任务（人力/组织/关键项目），在变革中兜底模糊性和不确定性。
