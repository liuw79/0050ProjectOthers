# 已发布文章存档

这个文件夹存放已完成并发布的文章。

## 文件命名规范

格式：`YYYY-MM-DD-文章标题.md`

示例：`2025-01-09-Claude-Code是最好的写作Agent.md`

## 文件内容

保留完整的文章内容（Markdown格式），可以在开头添加元数据：

```markdown
---
标题: [文章标题]
发布时间: YYYY-MM-DD
发布平台: [公众号/知乎/小红书]
阅读量: [如有]
点赞数: [如有]
评论数: [如有]
---

[文章正文]
```

## 用途

- 作为写作参考（学习风格）
- 避免重复内容
- 记录写作历史

---

最后更新：[日期]
