# 事业战略 邓良 9.21-22 上海 课程目录

## 一、 课程介绍与基础理论
1.  课程背景与“事业战略”的定义
2.  德鲁克事业理论引入：我们的事业是什么？将是什么？应该是什么？
3.  事业的定义：通过独特知识获得领先优势
4.  “独特知识”与“领先优势”的内涵与验证

## 二、 企业的独特知识与领先优势
1.  独特知识：企业无形的、底层的能力（案例：南孚电池）
2.  领先优势的三个判断维度：购买偏好、未来前景、经济绩效
3.  企业的基本功能：营销、创新、生产力

## 三、 企业的三项基本功能
1.  **营销**：核心是创造顾客，避免制造成本
2.  **创新**：核心是引领变化，避免滑向平庸
3.  **生产力**：核心是资源的放大，避免易受攻击

## 四、 外部环境分析与战略准备
1.  PEST分析与德鲁克的“和平时期”与“战时状态”
2.  战时状态下的经营策略：资源本地化、管制者作为顾客、重新审视机会
3.  未来市场环境的预判：做好迎接“严酷冬天”与“零增长管理”的准备

## 五、 企业组织与关系
1.  **劳资关系 vs 管理关系**：从金钱连接转向事业连接
2.  **管理者的五项基本活动**：设定目标、组织工作、激励沟通、绩效衡量、培育人才
3.  **目标管理与自我控制**：激发管理者的工作动机
4.  **组织精神（企业文化）**：强调对顾客的承诺与高标准
5.  **高层团队的有效性**：承担责任与决策

## 六、 事业健康度诊断与整体性管理
1.  企业健康状态的四种类型
2.  整体性管理：站在顾客角度审视企业
3.  组织诊断：站在贡献的角度（营销、创新、生产力）

## 七、 产品组合与创新策略
1.  **产品分类与对策**：
    *   今日的生机来源（释放资源）
    *   未来的增长来源（喂饱机会）
    *   昨日的生机来源（摆脱昨天）
    *   附带产品、开发中产品、问题型产品等
2.  **创新机遇**：识别“灰姑娘”产品（意外事件）
3.  **资源配置原则**：10%的产品创造90%的成果，应投入100%的精力

## 八、 行业壁垒与竞争战略
1.  **行业壁垒分析**：五力模型（潜在进入者、替代品、竞争对手等）
2.  **三场战役**：
    *   **仿品战**：核心在于渠道壁垒
    *   **新品战**：核心在于客户锁定（忠诚度）
    *   **技术战**：核心在于利基市场与技术迭代
3.  **流程组合**：明确哪些业务自营，哪些外包

## 九、 顾客三问与市场定位（战略起点）
1.  **第一问：顾客是谁？**
    *   现实顾客 vs 潜在顾客
    *   顾客的定义：由需求定义，谁决定购买
    *   选择潜在顾客的标准：稳定大量、能赢得竞争
2.  **第二问：顾客购买的是什么？**（未详细展开，隐含在价值讨论中）
3.  **第三问：顾客认为的价值是什么？**（未详细展开，隐含在价值讨论中）
4.  **市场与渠道**：市场是战场，渠道由顾客定义

## 十、 企业家精神与应对焦虑
1.  应对焦虑的建议：找到源泉（顾客价值）、暂时忘掉增长、沉浸当下
2.  利润的本质：为未来发展准备的成本（更新、淘汰、风险、不确定性）
3.  回归初心：重新思考事业的意义与价值
