# 夏凯：2026年度营销战略规划与增长路径

## 1. 课程基本信息
- **讲师**：夏凯（曾任用友营销学院创始院长，25年+大客户销售与营销管理经验）
- **核心主题**：年度营销战略规划、市场细分与洞察、组织绩效与执行落地
- **面向对象**：B2B企业创始人、营销VP、销售管理者

## 2. 核心金句
- **关于战略与选择**
    - “上兵伐谋，胜兵先胜。”——先找到必胜的市场和策略，再开战。
    - “你不选市场，市场就会选你。”——什么都想要，最后什么都得不到。
    - “动不动就降价是缺乏对经营深刻理解的表现。”（引用宋志平）
    - “少即是多，便宜即是贵。”——聚焦高价值客户，看似量少，实则利润与效率更高。
- **关于客户与价值**
    - “客户买什么不重要，客户把你当什么买才重要。”
    - “不要关注客户，要关注客户的客户。”——从终端需求倒推解决方案。
    - “没有高低贵贱之分，只有适用性。”——卖产品、卖方案、卖战略各有其生存之道，最怕错配。
- **关于执行与绩效**
    - “模式错配是最大的成本。”
    - “有枪便是王？不，利润才是王。”——销售额高不代表贡献大，要看费用和利润。
    - “以前老板说老板的，下属做下属的，最后口和心不合。”——规划的核心是达成深度战略共识。

## 3. 关键模型与方法论

### 3.1 GMS 销售/业务模式
根据客户与供应商的**投入程度**划分三种模式，避免错配：
- **S (Standard/Product) - 卖产品**：
    - **特征**：客户低投入（功能性需求，充饥），供应商低投入（标准化）。
    - **逻辑**：效率优先，标准化覆盖，走量。
    - **人才**：Feature（产品特性专家）/ Seller。
- **M (Marketing/Solution) - 卖方案**：
    - **特征**：客户中高投入（改善型需求，请客吃饭），供应商中高投入（定制化/组合）。
    - **逻辑**：价值优先，解决具体业务问题。
    - **人才**：Hunter（猎人）/ Farmer（农夫）。
- **G (Growth/Global/Strategy) - 卖战略**：
    - **特征**：双方高投入（战略性需求，为了未来/晋升/联姻）。
    - **逻辑**：共生共赢，深度绑定，解决未来发展问题。
    - **人才**：战略客户经理（需具备谋定而后动的策略能力）。

### 3.2 市场机会四象限（增长路径）
用于规划2026年增长点：
1. **老客户 + 老产品**：复购（难度：低，基石）。
2. **老客户 + 新产品**：交叉销售（价值最高，“老树开新花”，利润主要来源）。
3. **新客户 + 老产品**：新客户开发 / 竞对替换（难度：中）。
4. **新客户 + 新产品**：全新市场（难度：高，战略性投入）。
* **策略**：明确各象限占比，如老客户复购占30%，交叉销售占30%，新客户开发占40%。

### 3.3 利润增长六步路径
当面临亏损或需要增长时，不要只懂“加人”或“降价”，应从六个维度思考：
1. **增加销量**：最直接，但受限于市场容量。
2. **提高单价**：提供更高价值，提升品牌溢价。
3. **提高产品边际毛利**：优化供应链，降低BOM成本。
4. **降低边际成本**：减少销售过程中的变动成本。
5. **调整产品结构**：**核心策略**。砍掉负毛利/低毛利产品，聚焦高毛利产品（如海康规定低于2000万销售额的产品停售）。
6. **降低固定成本**：裁员、缩减办公场地等（最后的手段）。

### 3.4 销售能力/人才矩阵
- **双高人才**（资历高+专业高）：合伙人/压舱石。
- **高产人群**（资历中+专业高 或 资历高+专业中）：核心骨干，20%的人贡献80%业绩。
- **高危人群**：
    - 资历浅+专业高（空降兵）：容易水土不服，需6-9个月磨合。
    - 资历深+专业低（小白兔/老油条）：组织的阻力，需激活或淘汰。

## 4. 经典案例复盘

### 4.1 岳阳林纸（国企转型）
- **背景**：传统造纸，面临利润增长瓶颈，产品同质化。
- **破局点**：
    1. **重新定义行业**：从“制造/出版”转变为“文化服务/阅读体验”。
    2. **关注客户的客户**：研究中信出版社的读者（如王石）在飞机上阅读的体验（折光率、手感），与出版社共创解决方案。
    3. **GMS分层**：推出“天、月、山、湖”系列，高端打品牌（进中行、政府），中低端走量。
- **结果**：利润大幅提升，股价上涨，从卖纸变成卖文化载体解决方案。

### 4.2 某液压机外企（高端制造）
- **背景**：技术好、价格贵30%，面临国产低价恶性竞争（三一、中联等客户全员降本）。
- **困境**：不仅要降价，还要被当作备胎。
- **破局点**：**战略客户经营**。
    1. **洞察**：发现客户（三一/徐工）的核心战略是“出海”（北欧、中东）。
    2. **共创**：针对北欧极寒、中东沙漠等极端环境，主动提供防漏油、耐候性联合研发方案。
    3. **升维**：从“采购对销售”升级为“研发对研发”、“高管对高管”，帮助客户解决出海痛点。
- **结果**：避开了国内价格战，通过帮助客户成功出海，锁定了高价值订单。

### 4.3 邮储银行（金融科技服务）
- **背景**：农商行市场利润倒挂，市场萎缩。
- **选择**：放弃传统的农商行红海，**All-in 邮储银行**。
- **逻辑**：洞察到邮储银行从储蓄转型金融业务的迫切需求，且组织能力、网点下沉能力强。
- **结果**：实现60%增长，订单做不过来。**启示**：敢于放弃非战略市场，聚焦高增长潜力的细分赛道。

## 5. 执行落地建议

1. **市场洞察五看**：看行业趋势、看客户战略、看竞争格局、看自身优势、看机会窗。
2. **一张白纸的价值**：思考如何让一张A4纸卖出更高价格？（分客群、分场景、分价值）。
3. **营销规划三张表**：
    - **市场容量洞察表**：区域/行业客户总量、老客户数、竞对客户数、空白市场数。
    - **产出预测表**：基于客户名单的收入预测（而非拍脑袋的增长率）。
    - **产品策略表**：明确用什么产品去支撑目标（强推新产品/高毛利产品）。
4. **过程管理**：
    - 既然只有30%的客户有商机，就不要把时间平均分配给200个客户。
    - 重点管理**商机出现率**、**赢单率**、**单产**。
    - 建立SOP（销售兵法），将最佳实践标准化。
