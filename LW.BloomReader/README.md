# BloomReader - 阅读如花开

> 帮助阅读困难者的 H5 阅读应用

## 核心理念

句子**位置不变**（保持原书排版），但**显示顺序按理解逻辑**重排，像花一瓣一瓣展开。

## 目标用户

- 阅读障碍者
- 长文阅读困难者
- 需要逐步理解复杂文本的读者

## 显示逻辑（花开顺序）

按照理解难度递进的顺序显示句子，而非原文顺序：

1. **主题句/核心观点** - 先亮出来，建立框架
2. **举例说明** - 用具体例子支撑
3. **细节描写** - 补充细节
4. **修辞装饰** - 最后添加文采

## 句子拆分

每个句子拆成两部分，分步显示：

- **Core（核心）**：句子的主要意思
- **Decor（装饰）**：修辞、修饰语、过渡词

示例：
```
原文：即使从我们日常生活中最微不足道的小事来看，我们用我们所掌握的有关他的一切概念，来充实我们所见到的这个人的音容笑貌。

拆分：
- Core: 我们用我们所掌握的有关他的一切概念，来充实我们所见到的这个人的音容笑貌
- Decor: 即使从我们日常生活中最微不足道的小事来看，
```

## 视觉设计

- 未显示：文字颜色与背景相同（隐形）
- 核心句：深棕色，清晰醒目
- 主题句核心：加粗，更深色
- 修辞句：橄榄绿，区分度高
- 诗意句：斜体，柔和色调

## 交互方式

| 操作 | 效果 |
|------|------|
| 点击页面 / 空格键 | 显示下一部分 |
| 左箭头 / 点击左侧◀ | 上一跨页 |
| 右箭头 / 点击右侧▶ | 下一跨页 |
| 自动播放 | 定时自动显示 |
| 重置 | 回到初始状态 |

## 页面布局

- 模拟打开的书本，左右两页
- 进度条显示阅读进度
- 底部控制按钮
