# DOCX转Markdown工具使用说明

## 工具简介

这是一个专门用于将Word文档(.docx)转换为Markdown格式(.md)的Python工具。支持单文件转换和批量转换功能。

## 功能特点

- ✅ 支持.docx和.xml格式的Word文档
- ✅ 自动识别标题层级并转换为Markdown标题格式
- ✅ 支持表格转换为Markdown表格格式
- ✅ 支持单文件转换和批量转换
- ✅ 自动生成输出文件名或自定义输出路径
- ✅ 错误处理和转换状态反馈

## 安装依赖

工具依赖以下Python库：
```bash
pip install python-docx markdownify
```

## 使用方法

### 1. 转换单个文件
```bash
# 基本转换（自动生成.md文件）
python3 docx_to_md.py input.docx

# 指定输出文件名
python3 docx_to_md.py input.docx output.md
```

### 2. 批量转换目录
```bash
# 转换目录下所有.docx和.xml文件
python3 docx_to_md.py /path/to/directory
```

### 3. 查看帮助
```bash
# 显示使用说明
python3 docx_to_md.py
```

## 支持的格式转换

### 标题转换
- Word标题1 → `# 标题`
- Word标题2 → `## 标题`
- Word标题3 → `### 标题`
- Word标题4 → `#### 标题`
- Word标题5 → `##### 标题`
- Word标题6 → `###### 标题`

### 表格转换
Word表格会自动转换为Markdown表格格式：
```markdown
| 列1 | 列2 | 列3 |
| --- | --- | --- |
| 数据1 | 数据2 | 数据3 |
```

## 注意事项

1. **文件格式**：工具支持.docx和.xml扩展名的Word文档
2. **编码**：输出的Markdown文件使用UTF-8编码
3. **样式处理**：复杂的Word样式（如字体颜色、背景等）不会保留
4. **图片处理**：当前版本不支持图片转换
5. **路径要求**：建议使用相对路径或绝对路径

## 常见问题

### Q: 转换失败怎么办？
A: 检查以下几点：
- 文件是否存在且可读
- 文件是否为有效的Word文档格式
- 是否安装了所需的依赖库

### Q: 支持哪些Word版本？
A: 支持Word 2007及以后版本的.docx格式文档

### Q: 转换后的格式不理想怎么办？
A: 可以手动调整Markdown文件，或者在Word中调整文档结构后重新转换
