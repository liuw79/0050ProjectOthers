# 高维学堂组织设计工作坊准备材料

## 工作坊概述

**智邑组织设计工作坊**是一个为期2天1晚的学习体验，聚焦于"战略导向的组织设计，让企业战略DNA匹配最优组织管理体系"。本材料旨在帮助高维学堂团队充分准备，将工作坊内容与现有的CEO管理工具有机结合，实现最大化学习效果。

## 工作坊目标与高维学堂的对接点

| 工作坊目标 | 高维学堂对接点 | 准备材料 |
|---------|--------------|---------|
| 推进组织共同语境构建 | 基于OGSM的战略体系 | CEO战略仪表盘 |
| 战略-组织动态适配系统 | "3+N"产品体系与组织匹配 | CEO产品开发管理文档 |
| 打造高绩效团队组织领导力 | 战役制运营与学习小组机制 | CEO运营流程文档 |

## 参会团队建议

根据工作坊要求，每个企业的"战略-组织设计"团队需包含以下角色：

1. **高层（CEO/战略负责人）**：关注战略定位与组织效能
   - 携带材料：CEO战略仪表盘、CEO决策支持工具
   - 关注点：OGSM如何转化为组织设计

2. **执行层（COO/业务单元负责人）**：关注运营执行
   - 携带材料：CEO运营流程文档、CEO产品开发管理文档
   - 关注点：战役制运营体系如何影响组织架构

3. **赋能层（CHO/HRD）**：关注组织支持体系
   - 携带材料：CEO团队管理文档、人才矩阵管理表
   - 关注点：人才战略与组织发展规划

4. **建议增加（CMO/客户负责人）**：关注客户视角
   - 携带材料：CEO客户分析工具
   - 关注点：客户分层管理如何影响组织设计

## 各模块重点与高维学堂工具整合

### 模块1：定向链接（战略定位与组织定位）

**工作坊内容**：
- 企业的战略阶段定位与组织阶段定位
- 企业的价值链选择与组织架构定位
- 企业愿景与公司级组织效能确认

**高维学堂整合建议**：
1. 准备**OGSM执行追踪表**，展示"平台高级指标：稳定孵化创始人，发展产品系统，扩展服务能力，2025年实现GMV2.4亿"如何影响组织设计
2. 准备**S/A/B/C级客户分层管理表**，说明不同客户价值定位如何影响组织架构选择
3. 准备"创始人发展路径"图，展示价值链选择与组织需求的关系

**具体工具参考**：
```

|  |  | S2: 战役制运营 | 内容爆发指标10个 |
```


### 模块3：行动链接（构建组织行动 明确人才战略）

**工作坊内容**：
- 公司组织行动层面的人才标准确定
- 基于企业阶段的经验积累的核心人才胜任素质
- 基于战略需求的核心关键人才地图规划
- 核心人才的人才供应链规划设计

**高维学堂整合建议**：
1. 准备**人才九宫格**分析，展示高潜力人才管理方法
2. 准备**核心人才追踪表**示例，展示人才管理闭环
3. 准备**创始人准入机制表**，作为特殊人才管理的案例分享
4. **补充说明**：基于一线反馈（如阿左案例），识别关键岗位（如销售、客户服务）所需的核心能力（如快速需求理解、短平快方案推荐），并在人才标准、培训计划和辅导机制（如周一复盘会）中予以体现。

**具体工具参考**：

## 工作坊参与行动计划

### 准备阶段（工作坊前）

1. **团队协调会议**：
   - 时间：工作坊前1周
   - 内容：明确团队成员角色、准备材料分工
   - 责任人：CEO

2. **材料准备**：
   - 时间：工作坊前3天完成
   - 内容：根据上述各模块要求准备对应工具和案例
   - 责任人：各部门负责人

3. **预期目标设定**：
   - 明确希望在工作坊中解决的关键问题
   - 确定工作坊后的落地计划初步方向

### 参与阶段（工作坊中）

1. **关键问题记录**：
   - 设立专人负责记录工作坊中的重要观点和方法
   - 对标高维学堂现状，记录差距与改进点

2. **工具应用实践**：
   - 在每个模块练习中，尝试应用高维学堂已有的工具
   - 记录工具使用中的不足和改进方向

3. **案例分享准备**：
   - 针对高维学堂已有的实践，准备2-3个可分享的案例
   - 在适当环节主动分享，获取反馈

### 落地阶段（工作坊后）

1. **复盘会议**：
   - 时间：工作坊后1周内
   - 内容：总结学习成果，确定落地方向
   - 责任人：CEO+参与团队

2. **工具优化**：
   - 基于工作坊学习，对现有CEO管理工具进行优化
   - 重点关注组织设计与战略执行的匹配度

3. **90天行动计划**：
   - 制定组织优化的90天行动计划
   - 与年度OGSM执行计划对接，确保一致性

## 工作坊期望收获

1. 清晰理解高维学堂当前组织形态与战略需求的匹配度
2. 获取"3+N"产品体系对应的最优组织设计方案
3. 明确从商学院到商学平台转型过程中的组织演进路径
4. 掌握基于创始人分级的差异化组织能力构建方法
5. 建立战役制运营与常规组织管理的有效结合机制 
6. **新增**：设计或优化适应快速销售转化需求的**人才辅导与实践机制**。
