# 高维学堂2025年OGSM策略规划

> **重要提示**：本OGSM为年度初始规划。关于2025年中的战略复盘与调整，请参考 [2025年中期战略复盘与调整会议纪要](./2025年中期战略复盘与调整.md)。


## 目的 (Objective)

> **平台高级指标**：在平台逻辑指引下，激活创始人高频参与，以3大产品系列与入企延展服务，赋能新生代企业的产业升级与经营成长，2025年实现GMV2.4亿，成为中国新商学的领跑者。

## 目标指标 (Goals)

### G1：创始人活跃目标
**目标值**：稳定活跃的创始人≥1000人
- 稳定活跃：每季度出现在商学课或案例课至少1次

### G2：课题爆班率目标
**目标值**：指定课题爆班率≥80%
- 爆班标准：60-80人（复训除外），课题相应的爆班人数详见爆班课题目录
- 爆班率=指定课题的爆班数/指定课题的总开班数

### G3：课堂加购率目标
- **横向加购率**：≥50%（课类产品的加购）
- **纵向加购率**：≥30%（非课产品的加购）
- 课堂加购率=课堂现场及课后一周内有加购的企业数/本课堂总企业数

### G4：客户分层与GMV目标
**目标值**：通过对客户的分类分层运营达到GMV2.4亿
- 客户分层：4层客户S/A/B/C
- 产品分类：4类解决方案（课类：商学课+团队能力课+方案辅导班；非课类：入企服务）

### G5：平台建设目标
**目标值**：完成平台规划1.0（零人一会双百三系统四机制+区域作战模式打造）
- **零人**：业财系统零人工介入
- **一会**：平台规划&推行双周会
- **双百**：百万伙伴计划+辅导老师百万计划
- **三系统**：班主任助手升级到3.0，老师（课题）助手1.0，创始人和企业的报名学习系统
- **四机制**：客户流转机制、满意度保障机制、激励机制、生态伙伴赛马机制

## 策略 (Strategies)

### S1：创始人运营四件套，实现创始人稳定活跃
> *支持目标指标：G1（创始人活跃）*

#### 策略内容
1. 通过标准化、轻量级的校友会运营，解决大规模建联和日常活跃问题
2. 通过"三个卡"的设计，提升创始人的真实活跃
   - A.25800CEO进阶卡+B.98000企业学习卡+C.108000创始人畅学卡
3. 通过学习交流群来运营个人卡，实现交叉影响销课
   - 通过案例课+学习小组方式运营企业卡创始人，实现交叉影响销课
   - 通过畅学卡+私董会/私塾方式运营S类创始人（畅学卡为免费授予制，累计采购高维服务超300万，或企服组认定对象）
4. 通过机构/个人渠道拉新和个人卡转介绍机制，促进增量创始人的进入

#### 衡量指标(M)
| 指标名称 | 目标值 | 说明 |
|---------|-------|------|
| 创始人校友会覆盖率 | 100% | 目标创始人群体的校友会覆盖比例 |
| CEO进阶卡用户 | 1000人 | 进阶卡持有用户数量 |
| 企业学习卡新增用户 | 300人 | 企业学习卡新增用户数（区域承接） |
| 学习小组月度主题活动参与率 | 70% | 学习小组月度活动的参与比例 |
| 个人卡转推荐率 | 25% | 转推荐率=有成功推荐购买个人卡的数量÷个人包总购买量×100% |
| 渠道贡献会员数 | ≥200人 | 通过渠道获取的新增会员数量 |

#### 协同部门
- **主导部门**：创始人社区
- **配合部门**：各区域/行业

---

### S2：爆班战役
> *支持目标指标：G2（课题爆班率）*

#### 策略内容
- 设计团报权益，设立爆班项目小组，通过爆班操盘"5+1"方案+爆班作战会+全国PK激励机制，确保爆班达成
- 5指每个爆班课题标配的5个动作
- 1指每个爆班课题特殊的1个动作
- 团报权益：包桌价、周六晚作业小灶
- 爆班作战会：每周进度公示+卡点解决+优秀经验分享

#### 衡量指标(M)
| 指标名称 | 目标值 | 说明 |
|---------|-------|------|
| 爆班课题个数 | ≥10个 | 按季度更新，见爆班选题目录 |
| 爆班方案执行率 | 100% | 爆班方案按计划执行的比例 |
| 爆班作战会的卡点解决率 | 100% | 爆班过程中遇到问题的解决效率 |
| PK机制的有效性 | - | 激励机制对团队激励的效果评估 |
| 客户满意度 | 在现有基础上不下降 | 客户满意度维持或提升 |
| 团队课每班包桌的企业占比 | - | 团队课中包桌企业的比例 |
| 爆班优秀实践分享 | - | 优秀实践案例的分享数量和质量 |

#### 协同部门
- **主导部门**：开课支持组
- **配合部门**：各区域/行业

---

### S3：3+N产品矩阵，提供组合方案包
> *支持目标指标：G3（课堂加购率）, G4（客户分层与GMV）*

#### 策略内容
- 以3+N的产品矩阵为客户提供组合方案包（含行业方案包），重构课堂运营和转化流程
- 纵向-问题解决四层次地图
- 横向-课程关联地图
- 3+N产品矩阵：商学课+方案辅导班+团队训练课+入企服务

#### 衡量指标(M)
| 指标名称 | 目标值 | 说明 |
|---------|-------|------|
| 课堂重构后的运营SOP执行率 | - | 新课堂运营标准操作流程的执行比例 |
| 3+N组合方案包数量 | ≥18个 | 含18门商学课的方案包总数 |
| 课程地图配套率 | 100% | 课程之间关联地图的完成度 |
| 课后跟进率 | 原则100% | 课程结束后的用户跟进完成率 |

#### 协同部门
- **主导部门**：课堂重构打样项目（负责人：雯祯）
- **配合部门**：产品组（产品）+企服支持（售卖接口构建）+各区域/行业

---

### S4：创始人套餐+团队加速包服务
> *支持目标指标：G1（创始人活跃）, G3（课堂加购率）*

#### 策略内容
- 以创始人套餐+团队加速包，分别满足创始人和团队的学习需求
- 以学习小组和陪跑的方式，支撑企服伙伴主动定向服务S和A类客户（如跨境、餐饮两大行业）
- 学习小组每月提供CEO的目标管理和成长互助
- 学习小组活动产生的费用由参与学友AA

#### 衡量指标(M)
| 指标名称 | 目标值 | 说明 |
|---------|-------|------|
| S级用户企均收入 | 150万 | S级用户（约40+家）的平均收入 |
| A级用户企均收入 | 40万 | A级用户（约300+家）的平均收入 |
| 课的收入总额 | 1.1亿 | 课程类产品的总收入 |
| 非课的收入总额 | 1.3亿 | 非课程类产品的总收入 |
| 学习小组月度主题活动参与率 | 70% | 学习小组月度活动的参与度 |
| 组合式的行业方案数量 | 2个 | 已开发的行业方案包（餐饮+跨境） |

#### 协同部门
- **主导部门**：开口课+三个套餐（创始人社区+各区域/行业）
- **配合部门**：企业服务（企服支持组+各区域/行业）

---

### S5：业财流程优化与平台建设
> *支持目标指标：G5（平台建设）*

#### 策略内容
- 通过梳理现有业财流程，解决流程中的卡点，实现业财全自动化流程落地
- 通过搭建全公司业务看板，实现全业务流程数字可视化
- 项目/机制交付及宣导会-解决执行推广问题
- 平台机制双周会推动落地
- 双百计划由相关负责任部门（区域+产品）补充执行

#### 衡量指标(M)
| 指标名称 | 目标值 | 说明 |
|---------|-------|------|
| 老师人数 | 10 | 核心老师资源池规模 |
| 伙伴人数 | 10 | 核心合作伙伴数量 |
| 平台规划方案1.0定稿 | 约迭代10次 | 平台规划方案的迭代完善次数 |
| 机制推行的及时性 | - | 各项机制推行的及时程度 |
| 机制推行的成功率 | - | 机制推行的成功实施比例 |
| 系统开发或改造实现占比 | >10% | 按平台规划实现的系统改造比例 |
| 业财打通程度 | 从40%提升到80%以上 | 业财系统整合度的提升 |
| 人工介入程度 | <10% | 需要人工干预的流程比例目标 |
| 区域作战模式 | 三大区域 | 成功实施区域作战模式的区域数量 |

#### 协同部门
- **主导部门**：平台后台OGSM
- **配合部门**：效能后台+各区域（双百）
