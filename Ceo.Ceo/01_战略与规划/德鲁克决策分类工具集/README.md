# 德鲁克决策分类工具集

基于彼得·德鲁克《管理的实践》第28章决策分析原则设计的完整工具集。

## 核心理念

### 决策分类的根本目的
> "这样的分类能确保决策对于企业整体产生实质贡献，而不是牺牲整体利益解决眼前或局部的问题。因为这个分类方式乃是根据问题与企业整体目标，以及问题和个别单位目标之间的关联性，来筛选问题，强迫管理者从企业整体角度思考问题。"

### 替代方案制定的核心原则
> "有个不变的决策原则，就是必须针对每个问题，制定出各种选择方案，否则很容易陷入'两者择其一'，非此即彼的陷阱中。只有提出各种可供选择的替代方案，才能把基本假设提升到意识的层次，迫使自己检视这些方案，测试其效能。"

## 使用建议

### 新手入门路径
1. **阅读理念**：先理解德鲁克的核心理念（本README）
2. **学习案例**：通过案例分析了解实际应用
3. **掌握速查**：熟练使用速查卡进行快速分类
4. **深度分析**：使用分析工具处理复杂决策
5. **标准实践**：用实践模板建立企业标准流程

### 日常使用流程
1. **快速分类**：使用速查卡判断决策层级
2. **替代方案**：制定多种可行的解决方案
3. **深度分析**：对重要决策使用分析工具
4. **流程执行**：按照相应层级的决策流程执行
5. **跟踪记录**：使用模板记录决策过程和结果

## 关键原则

1. **整体视角**：每个决策都要服务于企业整体目标
2. **避免局部优化**：不为解决眼前问题而牺牲整体利益
3. **系统思考**：从企业整体角度思考问题
4. **突破二元思维**：避免"非此即彼"的陷阱
5. **挑战基本假设**：通过替代方案制定检视潜在假设
6. **包含不行动**："不采取行动"也是完整的决策选择

## 常见误区

1. **只看眼前**：忽视决策的长期影响
2. **孤立思考**：不考虑对其他部门的影响
3. **经验主义**：用过去的经验处理新问题
4. **级别错配**：小问题大决策，大问题小决策
5. **缺乏替代方案**：陷入"非此即彼"的思维陷阱
6. **忽视不行动**：没有考虑"什么都不做"的选择

## 实施建议

1. **建立标准**：在企业内部建立统一的决策分类标准
2. **培训团队**：让管理层掌握分类方法和替代方案制定
3. **定期回顾**：每季度回顾决策分类的准确性
4. **建立流程**：不同级别决策采用不同的审批流程
5. **记录学习**：建立决策案例库，积累经验
6. **文化建设**：鼓励基于事实的决策和合理的试错
