# 决策分类速查卡

## 30秒快速分类法（基于德鲁克四原则）

### 第一步：快速筛选（10秒）

**德鲁克四原则快速检查：**
1. **未来性**：这个决策的承诺时间超过1年吗？ □是 □否
2. **影响范围**：这个决策会影响3个以上部门吗？ □是 □否  
3. **品质考虑**：这个决策如果错了，代价很高且难以逆转吗？ □是 □否
4. **独特性**：这个决策是独特的，没有标准程序可循吗？ □是 □否

**快速判断：**
- 4个"是" → **战略决策**
- 3个"是" → **战略决策**
- 2个"是" → **管理决策** 
- 1个"是" → **管理决策**
- 0个"是" → **操作决策**

### 第二步：德鲁克四原则确认（10秒）

**未来性信号：**
长期承诺、战略转型、基础建设、人才培养、品牌建设

**影响范围信号：**
跨部门、全公司、客户影响、供应链、合作伙伴

**品质考虑信号：**
高投资、不可逆、声誉风险、法律风险、竞争优势

**独特性信号：**
首次尝试、创新项目、无先例、需要创造、突破性

### 第三步：综合评估（10秒）

**提升决策级别的信号：**
- 承诺时间超过预期
- 影响范围扩大
- 错误代价高昂
- 缺乏标准程序

**发现提升信号 → 上调决策级别**

### 第四步：替代方案快速检查（5秒）

**德鲁克替代方案原则：**
□ 是否考虑了至少3个不同方案？
□ 是否包含了"不行动"选项？
□ 是否避免了"非此即彼"的陷阱？
□ 是否挑战了基本假设？

**没有替代方案 → 暂缓决策，先制定方案**

## 决策级别对应表

| 决策级别 | 决策者 | 时间要求 | 流程复杂度 |
|---------|--------|----------|------------|
| **战略决策** | CEO/董事会 | 1-6个月 | 正式程序 |
| **管理决策** | 中高层管理 | 1-4周 | 标准流程 |
| **操作决策** | 部门主管 | 1-7天 | 简化流程 |

## 常见决策类型速查

### 战略决策（通常需要CEO/董事会）
- ✓ 进入新业务领域
- ✓ 重大投资或并购
- ✓ 商业模式调整
- ✓ 组织架构重大变革
- ✓ 品牌定位调整
- ✓ 核心产品线调整

### 管理决策（通常需要中高层）
- ✓ 新系统上线
- ✓ 跨部门流程优化
- ✓ 重要人事任免
- ✓ 年度预算调整
- ✓ 供应商重大变更
- ✓ 办公场所搬迁

### 操作决策（通常部门主管可决定）
- ✓ 日常政策调整
- ✓ 部门内部流程优化
- ✓ 常规采购决定
- ✓ 员工福利微调
- ✓ 工作标准修订
- ✓ 日常运营调整

## 紧急情况处理

### 紧急但重要的决策
**原则：**先按紧急程度处理，后补充正式流程

**步骤：**
1. 立即评估风险等级
2. 确定最小可行决策范围
3. 快速征求关键人员意见
4. 做出临时决策
5. 事后补充完整决策流程

### 决策升级机制
**什么时候需要升级？**
- 发现新的重大风险
- 影响范围超出预期
- 遇到意外阻力或反对
- 需要额外资源支持

## 决策质量检查清单

### 决策前检查
□ 问题定义清楚了吗？
□ 决策分类准确吗？
□ 决策者级别合适吗？
□ 相关人员都参与了吗？
□ 风险评估充分吗？

### 决策后检查
□ 决策传达到位了吗？
□ 执行计划明确了吗？
□ 监督机制建立了吗？
□ 调整机制准备了吗？
□ 学习总结安排了吗？

## 特殊情况处理

### 边界模糊的决策
**处理原则：**宁可高估级别，不可低估风险

**具体做法：**
1. 向上一级征求意见
2. 增加决策参与人员
3. 延长决策时间
4. 增加风险控制措施

### 跨级决策冲突
**处理原则：**以影响最大的维度为准

**解决步骤：**
1. 重新分析四个维度
2. 识别最关键的风险点
3. 按最高级别要求处理
4. 建立多级监督机制

## 德鲁克决策分类的核心理念

> "这样的分类能确保决策对于企业整体产生实质贡献，而不是牺牲整体利益解决眼前或局部的问题。因为这个分类方式乃是根据问题与企业整体目标，以及问题和个别单位目标之间的关联性，来筛选问题，强迫管理者从整个企业的观点来看自己的问题。" —— 德鲁克《管理的实践》

**核心要点：**
- 决策分类的目的：确保每个决策都服务于企业整体目标
- 避免局部优化：不为解决眼前问题而牺牲整体利益
- 系统思维：强迫管理者从企业整体角度思考问题
- 目标关联：考虑决策与企业整体目标的关联性
