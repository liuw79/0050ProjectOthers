# CEO 工作区文档结构

本项目用于存放 CEO 日常管理、战略规划、组织建设、激励设计等相关文档和工具。

## 目录结构概览

*   **`00_管理驾驶舱与工具集/`**: 包含 CEO 管理仪表盘相关的实施计划、模板、数据采集规范及决策支持工具等。
*   **`01_战略与规划/`**: 存放公司级战略文档，如年度 OGSM、管理工具总体说明、战略仪表盘指南等。
*   **`02_产品与运营/`**: 包含核心业务流程、产品开发管理相关的文档。
*   **`03_组织与团队/`**: 涉及团队管理、组织架构、人才发展、以及参与外部组织工作坊的相关材料。
*   **`04_客户与市场/`**: 存放客户分析、客户分层管理等相关工具和文档。
*   **`05_财务与监控/`**: 包含财务监控工具、现金流预测、预算管理等相关指南。
*   **`06_激励机制设计/`**: (新建) 用于系统性地设计和管理公司整体激励方案。
    *   包含原则、**绩效衡量体系 (如 `团队绩效指标库`)**、短期/长期激励、非物质激励、管理沟通和参考输入等子模块。
*   **`90_改进与记录/`**: 用于存放项目过程中的改进记录、思考或临时性文件。
*   **`organize_files.ps1`**: 用于整理文件目录的脚本 (可按需更新说明)。
*   **`README.md`**: (本文档) 项目的总体说明和目录索引。
