# GWXT经营分析会管理系统

## 项目概述
本系统用于支持GWXT每月经营分析会的问题发现、记录、跟踪和解决方案管理。

## 文档结构
```
├── README.md                    # 项目说明
├── 会议记录/                    # 每月会议记录
├── 问题跟踪/                    # 问题发现和跟踪
├── 解决方案库/                  # 解决方案数据库
├── 人员事件跟踪/                # 重要人员和事件管理
└── 模板/                       # 各类模板文件
```

## 使用说明
1. **会议记录**：每月会议后更新相应月份的记录文件
2. **问题跟踪**：发现问题时及时记录，定期更新状态
3. **解决方案库**：积累成功的解决方案，便于复用
4. **人员事件跟踪**：记录关键人员动态和重要事件

## 快速开始
- 查看 `模板/` 目录获取各类模板
- 使用 `问题跟踪/问题清单.md` 记录新问题
- 参考 `解决方案库/` 寻找类似问题的解决方案
