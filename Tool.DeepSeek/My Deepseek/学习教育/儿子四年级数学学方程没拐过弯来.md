## User

儿子四年级数学学方程没拐过弯来，他总是写x=一个式子，其实还是以前的解题思路，怎么由浅入深启发他呢？

## Assistant

帮助孩子从算术思维过渡到代数思维需要循序渐进，以下是分步骤的引导方法：

### 第一步：用生活实例建立变量概念

- 举例说明：“小明有10元钱，买铅笔花了一些后还剩4元。花了多少钱？”传统算术：10 - 4 = 6（直接计算答案）代数思维：设花的钱为x，写出方程 10 - x = 4，再解方程。关键点：让孩子对比两种方法，理解“用x代替未知数”的意义。

### 巩固练习模板

- 基础题（直接关系）：“一个数加5等于12，求这个数。”（x + 5 = 12 → x = 7）

- 进阶题（多步运算）：“爸爸年龄是女儿的4倍，两人年龄和是50岁，求女儿年龄。”（x + 4x = 50 → 5x = 50 → x = 10）
