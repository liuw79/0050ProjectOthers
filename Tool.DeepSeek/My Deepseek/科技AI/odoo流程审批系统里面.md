- 流程设计优化方案：节点合并：如果两个审批步骤的业务意义相近（如示例中的"离职证明"和"公章审批"），建议合并为一个审批节点，在审批意见中区分不同审批内容。动态审批人设置：使用domain过滤或compute方法动态设置审批人，确保同一审批人在不同阶段只需审批一次。

def action_approve(self):
    for record in self:
        # 检查后续节点是否同一审批人
        next_stage = record.get_next_stage()
        if next_stage.approver == record.env.user:
            record.write({
                'stage_id': next_stage.id,
                'approver': next_stage.approver.id
            })
    return super().action_approve()
