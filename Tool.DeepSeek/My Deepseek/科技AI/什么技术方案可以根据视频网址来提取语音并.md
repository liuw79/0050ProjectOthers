### 3. 语音转文字（ASR）

目的：将音频转换为文本。

- 方案一：云服务API（适合高精度、快速处理）Google Cloud Speech-to-Text：支持多语言、实时转录。python复制下载from google.cloud import speech_v1p1beta1 as speech

audio = speech.RecognitionAudio(uri="gs://bucket/audio.wav")
config = speech.RecognitionConfig(language_code="zh-CN")
response = client.recognize(config=config, audio=audio)Azure Cognitive Services / AWS Transcribe：类似功能，按用量付费。
