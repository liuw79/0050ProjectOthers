- 工具：GitHub Codespaces 或 Gitpod步骤：在手机浏览器打开 GitHub Codespaces 或 GitPod。创建或进入一个代码仓库的云端环境（自动启动 VS Code Web 版）。在云端 VS Code 中安装 GitHub Copilot 插件。使用触屏或外接键盘编写代码，Copilot 会自动提供建议。优势：完整功能，与桌面端体验一致。缺点：依赖网络，免费版有使用时长限制。
