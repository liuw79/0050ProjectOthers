### 一、Node.js 是什么？

- 一句话定义：Node.js 是一个基于 JavaScript 运行时环境，专门用于在服务器端运行 JavaScript 代码（原本 JavaScript 只能跑在浏览器里）。

一句话定义：Node.js 是一个基于 JavaScript 运行时环境，专门用于在服务器端运行 JavaScript 代码（原本 JavaScript 只能跑在浏览器里）。

- 核心机制：事件驱动（Event-Driven）：通过事件循环（Event Loop）处理异步操作（比如读写文件、网络请求）。非阻塞 I/O（Non-Blocking I/O）：当一个任务需要等待（比如数据库查询），Node.js 不会“卡住”线程，而是继续处理其他任务。单线程 + 异步：虽然只有一个主线程，但通过异步操作实现高并发（类似餐厅里一个服务员同时服务多桌客人）。

- 单线程 + 异步：虽然只有一个主线程，但通过异步操作实现高并发（类似餐厅里一个服务员同时服务多桌客人）。
