## Assistant

我们可以用以下步骤直接解释：

1. 从根的定义出发如果x=2是方程的根，说明当x代入2时等式成立。例如，最简单的方程就是直接写出来：x = 2（但这是一元一次方程）。

2. 如何升级为二次方程？要让方程变成二次的，我们需要引入一个“平方项”。一种直接的方法是：

- 从等式 x = 2 出发，两边同时平方：(x)2=(2)2(x)^2 = (2)^2(x)2=(2)2得到 x² = 4。

3. 为什么平方后原根依然成立？因为当x=2时，左边是2²=4，右边也是4，等式成立。所以平方后的方程 x²=4 仍然包含原来的根x=2。

4. 为什么会多出另一个根x=-2？平方运算有一个特点：正数和负数的平方结果都是正数。例如：

- 2² = 4，(-2)² = 4。因此，方程 x²=4 的解不仅是x=2，还包括x=-2。
