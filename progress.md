# 2026-01-29｜学习任务：《战略设计总纲》课程深度学习

- **学习目标**：深入理解刘绍荣《战略设计总纲》课程核心逻辑，为后续公众号文章积累素材。
- **核心知识点梳理**：
  1.  **核心矛盾**：战术勤奋掩盖战略懒惰（老板的通病）。
  2.  **组织与战略**：组织决定战略（组织特质必须与战略模式匹配，否则会导致“组织撕裂”）。
  3.  **3+3+1模型**：判断企业价值的七个维度（外部：时机/赛道/模式；内部：业务/组织/经营；核心：创始人）。
  4.  **KSF模型**：思考力、领导力、敏捷力的层级分布。
  5.  **第二增长曲线**：正确的时间（第一曲线拉满）+ 正确的方向（顾客需求四象限）。
  6.  **四难一缺**：聚焦难、抓住难、分解难、共识难；缺活力。
  7.  **三种底层模式**：供应链型（效率）、品牌型（创新）、集成解决方案型（专业），忌“三合一”。
- **素材库更新**：
  - **金句**：“一个企业最大的浪费是战略性的浪费”、“老板什么都想要，那就把他干掉”。
  - **案例**：医药网（资源聚焦ToB）、ZARA（快时尚+组织特质）、悦品餐饮（组织特质错配）、华宝新能源（战略集群选择）。
- **未来选题储备**：
