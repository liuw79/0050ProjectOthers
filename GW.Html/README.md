# 高维学堂流程型组织课100期庆祝H5页面

## 项目简介

这是为高维学堂流程型组织课100期里程碑制作的庆祝H5页面，用于表达对学友们的感谢，并提供5朵小红花积分奖励的领取功能。

- **辅助色**: #C3E7F4, #91D6F2, #004960

## 使用方法

### 本地预览
1. 直接用浏览器打开 `index.html` 文件
2. 或者启动本地服务器：
   ```bash
   # 使用Python
   python -m http.server 8000
   
   # 使用Node.js
   npx serve .
   
   # 使用PHP
   php -S localhost:8000
   ```

### 部署上线
1. 将所有文件上传到Web服务器
2. 确保服务器支持静态文件访问
3. 替换二维码为真实的领取链接

## 自定义配置

### 替换二维码
在 `index.html` 中找到 `.qr-placeholder` 区域，替换为真实的二维码图片：

```html
<div class="qr-container">
    <img src="your-qr-code.png" alt="领取二维码" style="width: 160px; height: 160px; border-radius: 10px;">
</div>
```

### 修改奖励数量
在 `index.html` 中修改花朵数量和文字描述：

```html
<div class="flower-display">
    <!-- 根据需要增减花朵图标 -->
    <div class="flower-icon">🌺</div>
    <!-- ... -->
</div>
<div class="reward-text">
    <h3>X朵小红花</h3> <!-- 修改数量 -->
</div>
```

### 自定义颜色
在 `styles.css` 的 `:root` 选择器中修改CSS变量：

```css
:root {
    --primary-color: #00A0EB; /* 主色调 */
    --light-gray: #EFEFEF;    /* 背景色 */
    /* ... 其他颜色变量 */
}
```
