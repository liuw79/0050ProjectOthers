# 金句提取 Skill

## 任务定位

从高维学堂课程内容中提取高价值金句，生成结构化的金句文档，存入对应课程目录。

---

## 金句筛选标准

### 优质金句特征
1. **洞察深刻**：揭示商业本质、管理规律、人性真相
2. **表达精炼**：言简意赅，一句话说透一个道理
3. **场景共鸣**：能让1~10亿规模企业老板产生强烈共鸣
4. **传播性强**：适合分享、引用、做成金句卡片
5. **原创独特**：非常见鸡汤，有讲师独特视角
6. **行动指引**：能指导实际经营决策

### 排除内容
- 过渡性语句（"接下来我们讲..."）
- 口语化填充词（"那个、这个、然后"）
- 常见鸡汤（"坚持就是胜利"）
- 上下文依赖过强的句子
- 纯案例描述（无观点提炼）

## 金句分类体系

| 类别 | 说明 | 示例方向 |
|------|------|----------|
| **认知升级** | 颠覆常规思维的洞察 | 关于商业本质的重新定义 |
| **增长战略** | 企业增长相关的核心观点 | 从1亿到10亿的关键跨越 |
| **组织管理** | 团队、文化、人才相关 | 高管团队搭建、授权边界 |
| **商业模式** | 盈利逻辑、价值创造 | 定价、竞争、护城河 |
| **决策智慧** | 创始人决策相关 | 取舍、时机、节奏把控 |
| **创业心法** | 创始人心态、格局 | 孤独、坚持、自我突破 |

## 工作流程

### Step 1: 课程知识学习

> **调用公共模块：** 执行总纲 CLAUDE.md 中的「知识内容提取」模块（K1-K4）

### Step 2: 金句标记与筛选

1. 通读课程内容，理解主题和核心论点
2. 标记所有潜在金句（宁多勿漏）
3. 按6大标准逐条筛选
4. 去重合并相似观点

### Step 3: 润色优化

| 原则 | 说明 |
|------|------|
| **保真** | 不改变原意，不过度演绎 |
| **去杂** | 删除口语化表达、重复词 |
| **提炼** | 长句拆短，突出核心观点 |
| **增力** | 适当使用对比、排比增强力度 |

**润色示例：**
- 原文："其实我觉得吧，就是说，一个企业从1个亿做到10个亿，最难的不是业务，是那个，是创始人自己的那个认知要升级"
- 润色后："企业从1亿到10亿，最难跨越的不是业务，是创始人的认知天花板"

### Step 4: 分类整理与输出

按分类体系整理金句，生成结构化文档。
