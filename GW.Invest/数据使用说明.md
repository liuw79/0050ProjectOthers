# 投资管理数据表使用说明

## 文件说明

### 1. 企业信息表.csv
- **用途**：记录合作企业的基本信息
- **字段说明**：
  - 企业名称：合作企业的名称
  - 联系人：企业对接人姓名
  - 联系电话：联系方式
  - 合作开始时间：首次合作日期
  - 备注：其他重要信息

### 2. 门店信息表.csv
- **用途**：记录各个门店的详细信息
- **字段说明**：
  - 门店编号：唯一标识符（如：兰A、三B）
  - 门店名称：门店的完整名称
  - 所属企业：归属的企业名称
  - 开店时间：门店开业日期
  - 当前状态：开店期（无分红）或运营期（有分红）
  - 地址：门店地址
  - 备注：其他信息

### 3. 投资记录表.csv
- **用途**：记录所有投资交易
- **字段说明**：
  - 投资日期：投资发生的日期
  - 门店编号：对应门店信息表的编号
  - 门店名称：门店名称（便于查看）
  - 投资金额：投资的资金数额
  - 投资类型：股权投资、债权投资等
  - 投资比例：占门店的股份比例
  - 备注：投资相关说明

### 4. 分红记录表.csv
- **用途**：记录每月分红情况
- **字段说明**：
  - 分红年月：格式为YYYY-MM（如：2024-03）
  - 门店编号：对应门店信息表的编号
  - 门店名称：门店名称（便于查看）
  - 分红金额：当月分红数额
  - 到账日期：分红实际到账日期
  - 分红类型：月度分红、年终分红等
  - 备注：分红相关说明

## 使用建议

1. **Excel导入**：可以将CSV文件导入Excel，创建多工作表的工作簿
2. **数据维护**：定期更新门店状态，及时录入分红记录
3. **数据分析**：可以基于这些数据计算投资回报率、累计收益等指标
4. **备份**：建议定期备份数据文件

## 后续扩展

- 可以添加更多字段，如门店面积、员工数量等
- 可以增加财务分析表，自动计算各种投资指标
- 为小程序开发做数据结构准备
