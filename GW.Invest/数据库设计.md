# 投资管理系统数据库设计

## 数据库表结构设计

### 1. 企业信息表 (companies)
```sql
CREATE TABLE companies (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    company_name VARCHAR(100) NOT NULL UNIQUE,
    contact_person VARCHAR(50),
    contact_phone VARCHAR(20),
    cooperation_start_date DATE,
    address TEXT,
    notes TEXT,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

### 2. 门店信息表 (stores)
```sql
CREATE TABLE stores (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    store_code VARCHAR(20) NOT NULL UNIQUE,
    store_name VARCHAR(100) NOT NULL,
    company_id INTEGER NOT NULL,
    opening_date DATE,
    status ENUM('开店期', '运营期', '暂停', '关闭') DEFAULT '开店期',
    address TEXT,
    area DECIMAL(10,2),
    notes TEXT,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (company_id) REFERENCES companies(id)
);
```

### 3. 投资记录表 (investments)
```sql
CREATE TABLE investments (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    investment_date DATE NOT NULL,
    store_id INTEGER NOT NULL,
    investment_amount DECIMAL(15,2) NOT NULL,
    investment_type ENUM('股权投资', '债权投资', '追加投资') DEFAULT '股权投资',
    investment_ratio DECIMAL(5,2),
    contract_number VARCHAR(50),
    notes TEXT,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (store_id) REFERENCES stores(id)
);
```

### 4. 分红记录表 (dividends)
```sql
CREATE TABLE dividends (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    dividend_year_month VARCHAR(7) NOT NULL,
    store_id INTEGER NOT NULL,
    dividend_amount DECIMAL(15,2) NOT NULL,
    received_date DATE,
    dividend_type ENUM('月度分红', '季度分红', '年终分红', '特别分红') DEFAULT '月度分红',
    status ENUM('待收', '已收', '延迟') DEFAULT '待收',
    notes TEXT,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (store_id) REFERENCES stores(id),
    UNIQUE KEY unique_store_month (store_id, dividend_year_month)
);
```

## 索引设计

```sql
-- 企业信息表索引
CREATE INDEX idx_companies_name ON companies(company_name);

-- 门店信息表索引
CREATE INDEX idx_stores_code ON stores(store_code);
CREATE INDEX idx_stores_company ON stores(company_id);
CREATE INDEX idx_stores_status ON stores(status);

-- 投资记录表索引
CREATE INDEX idx_investments_store ON investments(store_id);
CREATE INDEX idx_investments_date ON investments(investment_date);

-- 分红记录表索引
CREATE INDEX idx_dividends_store ON dividends(store_id);
CREATE INDEX idx_dividends_month ON dividends(dividend_year_month);
CREATE INDEX idx_dividends_status ON dividends(status);
```

## 视图设计

### 1. 投资汇总视图
```sql
CREATE VIEW investment_summary AS
SELECT 
    c.company_name,
    s.store_code,
    s.store_name,
    s.status,
    SUM(i.investment_amount) as total_investment,
    AVG(i.investment_ratio) as avg_ratio,
    MIN(i.investment_date) as first_investment_date
FROM companies c
JOIN stores s ON c.id = s.company_id
JOIN investments i ON s.id = i.store_id
GROUP BY c.id, s.id;
```

### 2. 分红汇总视图
```sql
CREATE VIEW dividend_summary AS
SELECT 
    c.company_name,
    s.store_code,
    s.store_name,
    COUNT(d.id) as dividend_count,
    SUM(d.dividend_amount) as total_dividends,
    AVG(d.dividend_amount) as avg_monthly_dividend,
    MAX(d.dividend_year_month) as latest_dividend_month
FROM companies c
JOIN stores s ON c.id = s.company_id
JOIN dividends d ON s.id = d.store_id
WHERE d.status = '已收'
GROUP BY c.id, s.id;
```

### 3. 投资回报分析视图
```sql
CREATE VIEW roi_analysis AS
SELECT 
    inv.company_name,
    inv.store_code,
    inv.store_name,
    inv.total_investment,
    COALESCE(div.total_dividends, 0) as total_dividends,
    COALESCE(div.total_dividends, 0) / inv.total_investment * 100 as roi_percentage,
    DATEDIFF(CURRENT_DATE, inv.first_investment_date) / 30 as investment_months
FROM investment_summary inv
LEFT JOIN dividend_summary div ON inv.store_code = div.store_code;
```

## 数据关系说明

1. **一对多关系**：
   - 企业 → 门店 (一个企业可以有多个门店)
   - 门店 → 投资记录 (一个门店可以有多次投资)
   - 门店 → 分红记录 (一个门店可以有多次分红)

2. **约束条件**：
   - 门店编号全局唯一
   - 同一门店同一月份只能有一条分红记录
   - 投资金额和分红金额必须大于0
   - 投资比例范围0-100%

3. **业务规则**：
   - 只有状态为"运营期"的门店才能产生分红
   - 分红记录的年月格式为YYYY-MM
   - 投资日期不能晚于当前日期
