我想开发一个企业内部的投资记录和管理的应用，先从文件或数据表手动操作开始，后续再考虑做成微信或飞书小程序。

目前已经投资了两家餐饮企业的门店，分别是：
1. 兰湘子：兰A、兰B、兰C
2. 三出山：三A、三B

每一家门店，除了开店期，后面会每个月给分红。

请你整理一下开发思路，先极简：

## 开发思路（极简版）

### 1. 业务需求分析
- **投资对象**：餐饮企业门店
- **当前投资**：兰湘子（3个门店）+ 三出山（2个门店）= 共5个门店
- **核心功能**：投资记录 + 分红管理
- **时间周期**：开店期（无分红）→ 运营期（月度分红）

### 2. 数据结构设计
#### 主要数据表：
1. **企业信息表**
   - 企业名称、联系方式、合作开始时间

2. **门店信息表**
   - 门店编号、门店名称、所属企业、开店时间、状态（开店期/运营期）

3. **投资记录表**
   - 投资日期、门店、投资金额、投资类型、备注

4. **分红记录表**
   - 分红年月、门店、分红金额、到账日期、备注

### 3. 实施步骤
#### 第一阶段：Excel手动管理
- [ ] 创建Excel工作簿，包含上述4个工作表
- [ ] 设计数据录入模板和统计报表
- [ ] 建立基础的数据验证规则

#### 第二阶段：数据库化
- [ ] 选择数据库（SQLite/MySQL）
- [ ] 创建数据表结构
- [ ] 开发简单的数据录入界面

#### 第三阶段：小程序开发
- [ ] 选择平台（微信/飞书小程序）
- [ ] 设计用户界面
- [ ] 开发核心功能模块
- [ ] 部署上线

### 4. 核心功能模块
1. **投资管理**：记录投资金额、时间、门店
2. **分红管理**：月度分红录入、统计分析
3. **报表统计**：投资回报率、累计收益等
4. **门店管理**：门店状态跟踪、基本信息维护
