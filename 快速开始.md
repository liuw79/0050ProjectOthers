# 快速开始指南

欢迎使用这个内容创作系统！这是一个基于Claude Code的自动化写作工作流。

## 第一步：理解系统架构

这个系统包含：

1. **两层判断机制**
   - 第一层：判断工作区（公众号/视频/其他）
   - 第二层：判断任务类型（新写作/修改/审校等）

2. **完整的写作流程**
   - 公众号写作：9步流程（从需求到发布）
   - 其他工作区：待定（可根据需要添加）

3. **核心原则**
   - Think Aloud透明化思考
   - 调研先行确保准确性
   - 选题讨论避免方向错误
   - 个人素材库降AI味

## 第二步：完善个人信息（重要！）

系统已经搭建好基础架构，但需要你提供个人信息才能发挥最佳效果。

### 必需完成（优先级高）

1. **个人信息** (`写作参考/个人信息.md`)
   - 你是谁、职业、专长
   - 写作领域和目标受众
   - 参考：`写作参考/README.md`

2. **风格指南** (`写作参考/风格指南.md`)
   - 语言特征、句式偏好
   - 禁用词汇清单
   - 参考：`写作参考/README.md`

3. **代表性文章** (`写作参考/代表作/`)
   - 2-3篇你最满意的文章
   - 用于AI学习你的风格

4. **个人素材库** (`个人素材库/`)
   - 真实经历、观点、案例
   - 可以是社交媒体动态、工作笔记等
   - 这是降AI味的核心！
   - 参考：`个人素材库/README.md`

### 推荐完成（优先级中）

5. **信息源清单** (`写作参考/信息源.md`)
   - 你常看的网站、媒体、博客

### 可选完成

6. **其他工作区的CLAUDE.md**
   - 如果你需要视频创作、其他类型的写作
   - 可以参考 `公众号写作/CLAUDE.md` 创建

## 第三步：开始第一篇文章

### 方法1: 有完整需求

直接告诉AI：

```
我要写一篇公众号文章，主题是[XXX]，目标受众是[XXX]，预期字数[XXX]。
```

AI会自动：
1. 保存brief
2. 搜索相关资料（如需）
3. 提供选题方向供你选择
4. 学习你的风格
5. 使用你的个人素材
6. 创作初稿
7. 三遍审校
8. 完成配图

### 方法2: 只有想法

告诉AI你的想法：

```
我想写一篇关于[XXX]的文章，但还没想好具体怎么写。
```

AI会帮你：
1. 明确需求
2. 调研背景信息
3. 提供多个选题方向
4. 你选择后再开始写作

### 方法3: 修改已有文章

```
我有一篇文章需要修改，文件路径是[XXX]，需要修改的地方是[XXX]。
```

AI会：
1. 读取原文
2. 理解修改需求
3. 提供修改方案
4. 执行修改

### 方法4: 审校/降AI味

```
这篇文章AI味太重，帮我审校一下，文件路径是[XXX]。
```

AI会：
1. 读取文章
2. 执行三遍审校（内容、风格、细节）
3. 删除套话、改写AI句式
4. 加入真实细节和个人态度

## 第四步：理解工作流程

### 公众号写作的9步流程

1. **理解需求 & 保存Brief**
2. **信息搜索与知识管理** ⭐
3. **选题讨论** ⭐ 必做
4. **创建协作文档**（如需测试/配图）
5. **学习风格**
6. **使用个人素材库** ⭐
7. **创作初稿**
8. **三遍审校** ⭐（内容、风格、细节）
9. **文章配图** ⭐

详细说明见：`公众号写作/CLAUDE.md`

### 核心原则（不可妥协）

- ❌ 绝不编造数据
- ❌ 绝不使用过时信息
- ❌ 绝不省略Think Aloud
- ❌ 绝不跳过用户确认（重要决策）

## 第五步：使用技巧

### 技巧1: 利用Think Aloud

AI会透明化思考过程，你可以随时打断、纠正方向。

### 技巧2: 选题讨论很重要

不要让AI直接写，先讨论选题。这避免了方向错误导致的大量返工。

### 技巧3: 个人素材库是核心

个人素材库越丰富、越真实，文章就越有"人味"，越不像AI写的。

建议：
- 定期更新个人素材库
- 记录真实经历、想法、数据
- 不要编造素材

### 技巧4: 三遍审校降AI味

AI会自动执行三遍审校，但你也可以要求额外审校：

```
这段话AI味太重，帮我改写一下：[引用原文]
```

### 技巧5: 灵活调整流程

如果任务简单/紧急，可以要求跳过某些步骤：

```
这篇文章比较简单，不需要调研，直接写吧。
```

但核心原则不能妥协（不编造数据、不用过时信息等）。

## 常见问题

### Q1: 我还没准备好个人信息，可以先用吗？

**A:** 可以，但效果会打折扣。AI会尽力写作，但没有你的风格和素材，文章会更像"标准AI文"。

建议：至少先准备 `个人素材库`，哪怕只有10-20条真实经历。

### Q2: AI会不会直接复制我的素材？

**A:** 不会。AI会改写成长文逻辑，调整语气，融入上下文。你的素材是原料，AI会加工。

### Q3: 我可以编一些素材吗？

**A:** 强烈不建议！AI会相信你的素材是真实的。如果你编造，最终文章会失真。

### Q4: 流程太复杂了，能简化吗？

**A:** 流程是可以灵活调整的。如果任务简单，直接说"这篇文章比较简单，快速写一下"，AI会简化流程。

但核心原则不能妥协（不编造数据、选题讨论等）。

### Q5: AI为什么要先讨论选题，不能直接写吗？

**A:** 选题讨论避免了方向错误。如果AI直接写，写了一半发现不是你想要的，就浪费了大量时间。讨论选题后，你可以及时纠正方向。

### Q6: 三遍审校是什么？

**A:**
- 第一遍：内容审校（逻辑、事实、结构）
- 第二遍：风格审校（删除AI味、加入人味）
- 第三遍：细节打磨（标点、排版、节奏）

这是降AI味的核心流程。
