# 30分钟HTML和CSS基础学习计划

## 学习目标
- 理解HTML的基本结构和常用标签
- 掌握CSS的基本语法和样式设置
- 能够创建一个简单的网页

## 时间分配

### 第1-10分钟：HTML基础

#### HTML是什么？
- HTML (HyperText Markup Language) 超文本标记语言
- 用于创建网页的标准标记语言
- 使用标签来描述网页内容的结构

#### 基本HTML结构
这是一个HTML文档的基本结构,让我详细解释每个部分:

#### 常用HTML标签
- `<h1>` 到 `<h6>`：标题标签
- `<p>`：段落标签
- `<div>`：容器标签
- `<span>`：行内容器
- `<a>`：链接标签
- `<img>`：图片标签
- `<ul>` 和 `<li>`：无序列表
- `<ol>` 和 `<li>`：有序列表

### 第11-20分钟：CSS基础

#### CSS是什么？
- CSS (Cascading Style Sheets) 层叠样式表
- 用于控制网页的外观和布局
- 将内容与样式分离

#### CSS语法
```css
选择器 {
    属性: 值;
    属性: 值;
}
```

#### CSS引入方式
1. **内联样式**：`<div style="color: red;">`
2. **内部样式**：在`<head>`中使用`<style>`标签
3. **外部样式**：链接外部CSS文件

#### 常用CSS属性
- `color`：文字颜色
- `background-color`：背景颜色
- `font-size`：字体大小
- `font-family`：字体类型
- `margin`：外边距
- `padding`：内边距
- `width` 和 `height`：宽度和高度
- `border`：边框

#### CSS选择器
- **元素选择器**：`p { }`
- **类选择器**：`.class-name { }`
- **ID选择器**：`#id-name { }`

## 学习建议

### 学习方法
1. **边学边练**：每学一个概念就立即实践
2. **多看多写**：查看其他网页的源代码学习
3. **循序渐进**：从简单到复杂，逐步提升

### 推荐资源
- MDN Web Docs (Mozilla开发者文档)
- W3Schools 在线教程
- 菜鸟教程

### 下一步学习方向
- CSS布局（Flexbox、Grid）
- 响应式设计
- JavaScript基础
- 前端框架（如Vue.js、React）
