## 常见问题

### Q: 为什么我的CSS样式没有生效？
A: 检查以下几点：
- CSS语法是否正确（分号、大括号）
- 选择器是否正确
- 是否有其他样式覆盖了你的样式
- 浏览器是否已经刷新

### Q: 如何让网页在不同设备上都好看？
A: 这涉及到响应式设计，可以：
- 使用相对单位（%、em、rem）而不是固定像素
- 使用媒体查询(@media)针对不同屏幕尺寸设置样式
- 使用Flexbox或Grid布局

### Q: 学完这30分钟后应该学什么？
A: 建议学习顺序：
1. 深入CSS布局（Flexbox、Grid）
2. 响应式设计
3. JavaScript基础
4. 前端框架（Vue.js、React等）
