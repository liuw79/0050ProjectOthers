🎯 什么是盒模型？

CSS盒模型是网页布局的基础概念。每个HTML元素都可以看作一个矩形的盒子，这个盒子由四个部分组成： 内容(Content)、内边距(Padding)、边框(Border)和外边距(Margin)。

Margin 外边距
Border 边框
Padding 内边距
Content 内容

⚖️ box-sizing 属性对比

box-sizing属性决定了元素的总宽度和高度如何计算：

content-box (默认)

宽度 = 内容宽度
总宽度 = 内容 + 内边距 + 边框

总宽度: 250px
width: 200px; padding: 20px; border: 5px solid #e74c3c; box-sizing: content-box; /* 总宽度 = 200 + 40 + 10 = 250px */

border-box (推荐)

宽度 = 总宽度
内容宽度 = 总宽度 - 内边距 - 边框

总宽度: 200px
width: 200px; padding: 20px; border: 5px solid #e74c3c; box-sizing: border-box; /* 总宽度 = 200px */ /* 内容宽度 = 200 - 40 - 10 = 150px */

💡 实用技巧和最佳实践

🌟 推荐的全局设置

在CSS开头添加这段代码，让所有元素使用border-box模型，布局更直观：

* { box-sizing: border-box; } /* 或者更精确的设置 */ html { box-sizing: border-box; } *, *::before, *::after { box-sizing: inherit; }

🎯 常见应用场景

• 响应式布局：使用百分比宽度时，border-box让计算更简单
• 表单设计：输入框的padding不会影响总宽度
• 卡片布局：确保所有卡片尺寸一致
• 网格系统：避免因padding导致的换行问题

🔍 盒模型调试技巧

在开发中遇到布局问题时，这些技巧能帮你快速定位问题：

/* 临时添加边框，可视化所有元素的盒模型 */ * { border: 1px solid red !important; } /* 或者使用outline，不影响布局 */ * { outline: 1px solid red !important; } /* 使用背景色区分不同区域 */ .debug { background: rgba(255, 0, 0, 0.1); /* 浅红色背景 */ }
