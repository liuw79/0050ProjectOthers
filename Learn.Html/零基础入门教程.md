# HTML零基础入门教程

## 第一步：什么是HTML？

HTML就像是网页的"骨架"，它告诉浏览器如何显示内容。

想象一下：
- 如果网页是一本书，HTML就是书的目录和章节结构
- 如果网页是一栋房子，HTML就是房子的框架

## 第二步：HTML的基本概念

### 标签（Tag）
HTML使用"标签"来标记不同的内容。标签用尖括号包围：
```
<标签名>内容</标签名>
```

例如：
- `<h1><b>这是标题</b></h1>` - 创建一个大标题
- `<p>这是段落</p>` - 创建一个段落

### 重要规则
1. 大部分标签都有开始和结束：`<标签>` 和 `</标签>`
2. 结束标签前面有斜杠 `/`
3. 标签可以嵌套（一个标签里面包含另一个标签）

## 第三步：创建你的第一个网页

让我们从最简单的开始！

### 步骤1：基本结构
每个HTML文件都需要这个基本结构：

```html
<!DOCTYPE html>
<html>
<head>
    <title>我的第一个网页</title>
</head>
<body>
    <h1>你好，世界！</h1>
</body>
</html>
```

### 各部分解释：
- `<!DOCTYPE html>` - 告诉浏览器这是HTML5文档
- `<html>` - 整个网页的容器
- `<head>` - 网页的"头部"，包含不显示的信息
- `<title>` - 网页标题（显示在浏览器标签上）
- `<body>` - 网页的"身体"，包含所有可见内容
- `<h1>` - 最大的标题

## 第四步：常用标签学习

### 标题标签
```html
<h1>最大标题</h1>
<h2>二级标题</h2>
<h3>三级标题</h3>
<h4>四级标题</h4>
<h5>五级标题</h5>
<h6>最小标题</h6>
```

### 段落标签
```html
<p>这是一个段落。段落会自动换行。</p>
<p>这是另一个段落。</p>
```

### 文字格式
```html
<strong>粗体文字</strong>
<em>斜体文字</em>
```

### 链接
```html
<a href="https://www.baidu.com">点击访问百度</a>
```

### 图片
```html
<img src="图片地址" alt="图片描述">
```

### 列表
```html
<!-- 无序列表（带圆点） -->
<ul>
    <li>第一项</li>
    <li>第二项</li>
    <li>第三项</li>
</ul>

<!-- 有序列表（带数字） -->
<ol>
    <li>第一步</li>
    <li>第二步</li>
    <li>第三步</li>
</ol>
```

## 第五步：实践练习

现在让我们创建一个简单的个人介绍页面：

```html
<!DOCTYPE html>
<html>
<head>
    <title>我的个人介绍</title>
</head>
<body>
    <h1>关于我</h1>
    <p>你好！我是一名HTML初学者。</p>
    
    <h2>我的爱好</h2>
    <ul>
        <li>阅读</li>
        <li>音乐</li>
        <li>编程</li>
    </ul>
    
    <h2>联系我</h2>
    <p>邮箱：<a href="mailto:your@email.com">your@email.com</a></p>
</body>
</html>
```

## 第六步：如何查看你的网页

1. 将代码保存为 `.html` 文件（比如 `my-page.html`）
2. 双击文件，它会在浏览器中打开
3. 你就能看到你的第一个网页了！

## 第七步：常见错误和解决方法

### 错误1：标签没有关闭
❌ 错误：`<h1>标题`
✅ 正确：`<h1>标题</h1>`

### 错误2：标签嵌套错误
❌ 错误：`<h1><p>内容</h1></p>`
✅ 正确：`<h1><p>内容</p></h1>`

### 错误3：忘记引号
❌ 错误：`<a href=链接地址>文字</a>`
✅ 正确：`<a href="链接地址">文字</a>`

## 第八步：下一步学习什么？

掌握了基础HTML后，你可以学习：
1. **CSS** - 让网页变得漂亮（颜色、字体、布局）
2. **更多HTML标签** - 表格、表单等
3. **JavaScript** - 让网页有交互功能
