这是我创建的第一个HTML页面。HTML是网页的骨架，它告诉浏览器如何显示内容。

重要提示：HTML使用标签来标记不同类型的内容。每个标签都有开始标签和结束标签。

HTML基础知识

HTML的基本组成：

  • 标签（Tags）：用尖括号包围的关键词，如 <h1>、<p>
  • 元素（Elements）：开始标签 + 内容 + 结束标签
  • 属性（Attributes）：为元素提供额外信息

创建网页的步骤：

  1. 创建HTML文件（以.html结尾）
  2. 编写HTML代码
  3. 在浏览器中打开文件
  4. 查看效果并调整

HTML代码结构

<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <title>页面标题</title>
</head>
<body>
    <h1>这是标题</h1>
    <p>这是段落内容</p>
</body>
</html>

重要概念：

  • <!DOCTYPE html>：告诉浏览器这是HTML5文档
  • <html>：根元素，包含整个页面
  • <head>：头部，包含元信息（不显示在页面上）
  • <body>：主体，包含所有可见内容
  • <title>：页面标题（显示在浏览器标签上）
