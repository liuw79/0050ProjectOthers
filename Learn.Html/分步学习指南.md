# HTML分步学习指南

## 🎯 学习目标
通过5个简单步骤，让你从零开始学会创建网页。

---

## 📋 第一步：理解HTML是什么

### 什么是HTML？
- HTML = HyperText Markup Language（超文本标记语言）
- 它是网页的"骨架"，告诉浏览器如何显示内容
- 就像写文章时的标题、段落、列表一样

### 为什么要学HTML？
- 所有网页都是用HTML构建的
- 学会HTML，你就能创建自己的网页
- 这是学习网页开发的第一步

**✅ 完成标志**：理解HTML的作用

---

## 📋 第二步：学习HTML基本结构

### HTML文档的基本框架
```html
<!DOCTYPE html>
<html>
<head>
    <title>页面标题</title>
</head>
<body>
    页面内容
</body>
</html>
```

### 各部分说明
- `<!DOCTYPE html>` - 告诉浏览器这是HTML5
- `<html>` - 整个网页的容器
- `<head>` - 网页信息（不显示在页面上）
- `<title>` - 网页标题（显示在浏览器标签）
- `<body>` - 网页内容（显示在页面上）

### 🎯 实践任务
1. 打开记事本或任何文本编辑器
2. 复制上面的基本框架
3. 将"页面标题"改为"我的第一个网页"
4. 将"页面内容"改为"你好，世界！"
5. 保存为 `test.html`
6. 双击文件在浏览器中打开

**✅ 完成标志**：能在浏览器中看到"你好，世界！"

## 📋 第三步：学习常用标签

### 标题标签
```html
<h1>最大标题</h1>
<h2>二级标题</h2>
<h3>三级标题</h3>
```

### 段落标签
```html
<p>这是一个段落。</p>
<p>这是另一个段落。</p>
```

### 文字格式
```html
<strong>粗体文字</strong>
<em>斜体文字</em>
```

### 🎯 实践任务
修改你的 `test.html` 文件：
```html
<!DOCTYPE html>
<html>
<head>
    <title>我的第一个网页</title>
</head>
<body>
    <h1>欢迎来到我的网页</h1>
    <p>这是我的<strong>第一个</strong>网页。</p>
    <p>我正在学习<em>HTML</em>！</p>
</body>
</html>
```

**✅ 完成标志**：页面显示不同大小的标题和格式化文字

## 📋 第四步：添加链接和列表

### 链接标签
```html
<a href="https://www.baidu.com">访问百度</a>
```

### 列表标签
```html
<!-- 无序列表 -->
<ul>
    <li>项目1</li>
    <li>项目2</li>
    <li>项目3</li>
</ul>

<!-- 有序列表 -->
<ol>
    <li>第一步</li>
    <li>第二步</li>
    <li>第三步</li>
</ol>
```

### 🎯 实践任务
在你的网页中添加：
```html
<h2>我的爱好</h2>
<ul>
    <li>阅读</li>
    <li>音乐</li>
    <li>编程</li>
</ul>

<h2>有用的链接</h2>
<p>学习资源：<a href="https://www.w3schools.com">W3Schools</a></p>
```

**✅ 完成标志**：页面显示列表和可点击的链接

## 🆘 遇到问题怎么办？

### 常见错误
1. **页面显示代码而不是效果**
   - 检查文件是否保存为 `.html` 格式
   - 确保用浏览器打开，不是文本编辑器

2. **标签不起作用**
   - 检查标签是否正确关闭：`<h1>标题</h1>`
   - 检查是否有拼写错误

3. **链接点击无效**
   - 检查href属性是否有引号：`href="链接地址"`
   - 确保链接地址正确

### 调试技巧
- 在浏览器中按 `F12` 打开开发者工具
- 右键点击页面选择"查看页面源代码"
- 一次只添加一个标签，逐步测试
