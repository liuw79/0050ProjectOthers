# 协作文档文件夹

这个文件夹存放需要用户配合完成的任务清单。

## 什么时候创建协作文档？

当写作任务需要以下配合时：
- 真实测试/使用产品
- 提供配图/截图
- 提供特定数据或信息
- 其他需要用户配合的任务

## 文件命名规范

格式：`文章主题-协作清单.md`

示例：`Claude-Code教程-协作清单.md`

## 文件内容规范

```markdown

创建时间：YYYY-MM-DD

---

1. [ ] 步骤1
2. [ ] 步骤2
3. [ ] ...

**使用的Prompt（如需）：**
\`\`\`
[具体Prompt内容]
\`\`\`

**数据记录：**

|--------|------|------|
```
