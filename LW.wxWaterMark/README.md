# 证件水印小程序

一个简单的微信小程序，用于给上传的证件图片添加自定义水印。

## 功能特性

*   **图片上传**: 支持从手机相册选择图片或使用相机拍照。
*   **历史图片管理**: 
    *   自动保存上传过的图片（如身份证等），最多保存10张
    *   页面顶部横向滚动显示历史图片
    *   点击历史图片快速使用，无需重复上传
    *   长按历史图片可以删除
*   **自定义水印**: 
    *   提供 "仅供____使用不得用于其他用途，有效期至YY/MM/DD" 的模板。
    *   用户可以自由编辑 "____" 部分的用途文字。
    *   有效期自动计算为从今天起一个月后，并以 YY/MM/DD 格式显示。
*   **记住用途**: 自动保存上次输入的用途文字，下次打开时自动填充。
*   **水印效果**: 以半透明、倾斜角度45度将完整水印文字多行重复绘制在图片上，避免遮挡重要内容。
*   **保存到相册**: 生成带水印的图片后，可以直接保存到手机相册（需要用户授权）。

## 如何使用

1.  打开 "证件水印" 小程序。
2.  **选择图片**：
    *   如果有历史图片，可以在顶部横向滚动区域点击历史图片快速选择
    *   或点击图片预览区域的 "点击选择图片"，从相册选择或相机拍摄新图片
3.  图片选择成功后，会在预览区显示，并自动保存到历史（最多10张）。
4.  在图片预览下方的文本框中编辑水印文字，将 "____" 替换为实际用途（如"身份验证"）。
5.  编辑完成后，点击 "生成水印图片" 按钮。
6.  小程序会处理图片，添加45度斜向多行水印，并在成功后提示 "已保存到相册"。
7.  (首次保存时) 如果弹出授权请求，请允许小程序访问相册。
8.  完成后，你可以在手机相册中找到带有水印的图片。

**小贴士**：
*   长按历史图片可以删除
*   历史图片如果失效（临时文件过期），选择时会自动删除

## 开发设置

*   使用 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html) 导入项目即可进行开发和调试。

## 注意事项

*   为了在界面上完整显示各种尺寸的图片，预览图的最大高度被限制为屏幕高度的40%。
*   生成的图片质量被设置为 0.9（较高质量），可以在 `pages/index/index.js` 的 `wx.canvasToTempFilePath` 调用中修改 `quality` 参数调整。
