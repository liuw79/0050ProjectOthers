# 内容创作系统

基于Claude Code的自动化写作工作流，帮助你高效创作高质量内容。

## 系统特点

- ✅ **两层判断机制**：自动识别工作区和任务类型
- ✅ **完整的写作流程**：从需求到发布的9步流程
- ✅ **降AI味核心**：使用个人素材库，让文章更有人味
- ✅ **质量保证**：三遍审校机制，系统化提升文章质量
- ✅ **透明化协作**：Think Aloud原则，掌控AI的思考过程

## 快速开始

**如果你是第一次使用，请阅读：[快速开始.md](./快速开始.md)**

### 三步上手

1. **完善个人信息**（必需）
   - 创建 `写作参考/个人信息.md`
   - 创建 `写作参考/风格指南.md`
   - 在 `写作参考/代表作/` 放入2-3篇你的文章
   - 在 `个人素材库/` 放入你的真实经历和观点

2. **告诉AI你的需求**
   ```
   我要写一篇公众号文章，主题是[XXX]...
   ```

3. **跟随AI的引导**
   - AI会透明化思考过程
   - AI会提供选题方向供你选择
   - AI会使用你的素材创作
   - AI会三遍审校降低AI味

## 公众号写作流程（9步）

1. **理解需求 & 保存Brief**
2. **信息搜索与知识管理** ⭐
3. **选题讨论** ⭐ 必做
4. **创建协作文档**（如需测试/配图）
5. **学习风格**
6. **使用个人素材库** ⭐
7. **创作初稿**
8. **三遍审校** ⭐（内容、风格、细节）
9. **文章配图** ⭐

详细说明见：[公众号写作/CLAUDE.md](./公众号写作/CLAUDE.md)

## 核心原则（不可妥协）

- ❌ 绝不编造数据
- ❌ 绝不使用过时信息
- ❌ 绝不省略Think Aloud
- ❌ 绝不跳过用户确认（重要决策）

## 使用示例

### 示例1：写一篇新文章

```
我要写一篇公众号文章，主题是"Claude Code使用心得"，
目标受众是开发者，预期3000字左右。
```

AI会：
1. 保存brief
2. 搜索Claude Code的最新信息
3. 提供3-4个选题方向
4. 等你选择后，学习你的风格
5. 从你的个人素材库中搜索相关经历
6. 创作初稿
7. 三遍审校（删除AI味、加入人味）
8. 完成配图

### 示例2：修改已有文章

```
我有一篇文章需要降低AI味，文件路径是 公众号写作/草稿.md
```

AI会：
1. 读取文章
2. 执行三遍审校
3. 删除套话、改写AI句式
4. 加入真实细节和个人态度

### 示例3：快速咨询

```
Claude Code和Cursor有什么区别？
```

AI会：
1. 搜索最新信息
2. 给出对比分析
3. 快速回答（不走完整写作流程）

## 常见问题

### Q: 我还没准备好个人信息，可以先用吗？

A: 可以，但效果会打折扣。建议至少先准备 `个人素材库`，哪怕只有10-20条真实经历。

### Q: 个人素材库是什么？为什么重要？

A: 个人素材库存放你的真实经历、观点、案例。这是降AI味的核心！

AI会从你的素材库中搜索相关内容，用真实经历替代AI腔。

详见：[个人素材库/README.md](./个人素材库/README.md)

### Q: AI会不会直接复制我的素材？

A: 不会。AI会改写成长文逻辑，调整语气，融入上下文。

### Q: 流程太复杂了，能简化吗？

A: 流程是可以灵活调整的。如果任务简单，直接说"快速写一下"，AI会简化流程。但核心原则不能妥协。

### Q: 为什么要先讨论选题？

A: 避免方向错误导致的大量返工。讨论选题后，你可以及时纠正方向。
