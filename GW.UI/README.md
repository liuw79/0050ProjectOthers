# 现代化个人中心页面设计

## 项目概述

这是一个基于高维学院品牌规范设计的现代化个人中心页面，采用扁平化设计理念，提供高质量、优雅、和谐且具有视觉吸引力的用户界面。

## 设计特色

### 🎨 品牌色彩规范

严格遵循高维学院品牌指导原则：

- **主要颜色 (40%)**
  - 品牌主题色：RGB(0, 160, 235) - `#00A0EB`
  - 用于主要按钮、导航栏、关键操作、品牌标识

- **中性颜色 (50%)**
  - 浅灰色：RGB(239, 239, 239) - `#EFEFEF`
  - 深黑色：RGB(35, 24, 21) - `#231815`
  - 用于背景、卡片基础、分割线、文本

- **辅助颜色 (8%)**
  - 浅蓝色：`#C3E7F4`
  - 中浅蓝色：`#91D6F2`
  - 用于次要按钮、图标填充、进度条、标签

- **强调颜色 (2%)**
  - 浅米色：`#ECC2AD`
  - 深蓝色：`#2D459A`
  - 用于警告提醒、高亮标签、临时状态

### ✨ 设计亮点

1. **现代扁平化设计**

## 技术规格

### 字体规范
- **中文字体**: Alibaba PuHuiTi (优先使用粗体和常规字重)
- **英文字体**: Gotham (优先使用Gotham-Black和Medium字重)
- **备用字体**: PingFang SC, Inter, 系统默认字体

### 尺寸标准
- **品牌标识**: 最小宽度 ≥15mm (按比例转换为像素)
- **口号标识**: 最小宽度 ≥16mm
- **组合标识**: 最小宽度 ≥26.8mm

### 色彩模式
- 数字显示专用RGB值
- 避免使用CMYK或PANTONE
- 保持颜色一致性

## 设计理念

本设计遵循"以用户为中心"的设计原则，追求：

- **简洁性**: 去除冗余元素，突出核心功能
- **一致性**: 统一的视觉风格和交互模式
- **可用性**: 直观的导航和操作流程
- **美观性**: 现代化的视觉呈现和品牌表达
- **可访问性**: 支持各种用户群体的使用需求
