# GTM产品市场协同作战 - AI辅助学习系统

## 📋 项目概述

本项目是一个基于GTM（Go-To-Market）产品市场协同作战理念的AI辅助学习系统，旨在帮助学员深入理解产品上市与生命周期管理的核心概念，并通过AI技术辅助完成相关作业和练习。

### 🎯 项目目标

- **知识体系化**：整理GTM课件知识点，建立完整的知识框架
- **AI辅助学习**：利用AI技术辅助学员完成作业和练习
- **协同工具**：提供在线表格系统支持多用户协同编辑
- **实战演练**：通过工具表和案例学习提升实战能力

## 🏗️ 核心概念框架

### GTM产品市场协同作战

**GTM（Go-To-Market）**是连接IPD（集成产品开发）和IPMS（集成产品管理系统）的关键桥梁，强调产品、市场、销售等部门的紧密协同合作。

```
IPD → GTM → IPMS
产品开发 → 上市策略 → 产品管理
```

### 三大核心概念

1. **集成操盘的目的**：让产品从一开始就"对症下药"，确保商业成功
2. **产品上市与生命周期管理**：产品从诞生到退市的全过程管理
3. **GTM产品市场协同作战**：团队作战精神，实现产品大卖

## 🛠️ 技术架构

### 系统组件

- **前端界面**：HTML/CSS/JavaScript构建的现代化Web界面
- **后端API**：Node.js + Express + PostgreSQL
- **数据库**：PostgreSQL存储表格数据和配置
- **实时协作**：WebSocket支持多用户实时编辑
- **AI集成**：支持AI辅助作业和内容生成

## 🚀 快速开始

### 环境要求

- Node.js >= 14.0.0
- PostgreSQL数据库
- 现代浏览器（支持ES6+）

### 安装步骤

1. **克隆项目**
   ```bash
   git clone [项目地址]
   cd GW.GTM
   ```

2. **安装依赖**
   ```bash
   npm install
   ```

3. **配置数据库**
   ```bash
   # 创建数据库
   createdb gtm_assistant
   
   # 导入数据库结构
   psql gtm_assistant < database_schema_v2.sql
   ```

4. **启动服务**
   ```bash
   # 启动API服务
   npm start
   
   # 或开发模式
   npm run dev
   ```

5. **访问系统**
   - 打开浏览器访问：`http://localhost:3002`
   - 或直接打开 `gtm_dashboard.html`
