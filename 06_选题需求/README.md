# Briefs 文件夹

这个文件夹存放所有写作任务的brief。

## 文件命名规范

- 商单：`项目名-商单brief.md`
- 自发写作：`主题-brief.md`
- 示例：`Claude-Code-商单brief.md` 或 `AI写作工具对比-brief.md`

## Brief应包含内容

```markdown
# [文章主题] - Brief

## 基本信息
- 创建时间：YYYY-MM-DD
- 截稿时间：YYYY-MM-DD
- 预期字数：XXXX字
- 发布平台：[公众号/知乎/小红书等]

## 核心需求
[描述写作的核心目的和要传达的信息]

## 目标受众
[谁会读这篇文章]

## 关键信息
- 必须提及的内容：[列表]
- 必须避免的内容：[列表]

## 特殊要求
- [ ] 需要配图（X张）
- [ ] 需要真实测试
- [ ] 需要数据支撑
- [ ] 其他：[补充]

## 参考资料
- [链接或文件路径]
```
