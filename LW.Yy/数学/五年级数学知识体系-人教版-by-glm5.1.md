## 上册

### 第一单元：小数乘法

**学习目标**：掌握小数乘法的算理和算法，能正确进行小数乘法运算。

| 节次 | 知识点 | 核心内容 |
|------|--------|----------|
| 1.1 | 小数乘整数 | 意义：求几个相同加数的和。方法：小数→整数→算积→定小数点 |
| 1.2 | 小数乘小数 | 意义：求这个数的几分之几。积的小数位数 = 因数小数位数之和 |
| 1.3 | 积的近似数 | 四舍五入法、进一法、去尾法（根据实际场景选择） |
| 1.4 | 运算定律推广 | 交换律、结合律、分配律 → 从整数推广到小数 |
| 1.5 | 连乘·乘加·乘减 | 运算顺序与整数四则运算相同 |

**核心公式/规律**：
- 计算方法：先按整数乘法算，再看因数共有几位小数，从积的右边起数几位点上小数点
- 积的变化规律：一个数(0除外) × >1的数 → 积变大；× <1的数 → 积变小
- 运算定律：
  - 乘法交换律：a×b = b×a
  - 乘法结合律：(a×b)×c = a×(b×c)
  - 乘法分配律：(a+b)×c = a×c + b×c

**易错点**：
- 小数点位置（位数不够时用0补位）
- 末尾的0要化简去掉
- 乘法分配律的逆向运用：a×c + b×c = (a+b)×c

### 第三单元：小数除法

**学习目标**：掌握小数除法的算理和算法，理解循环小数。

| 节次 | 知识点 | 核心内容 |
|------|--------|----------|
| 3.1 | 小数除以整数 | 按整数除法去除，商的小数点与被除数对齐 |
| 3.2 | 一个数除以小数 | 除数→整数（被除数同步扩大），再按整数除法做 |
| 3.3 | 商的近似数 | 根据需要四舍五入（保留到分/角） |
| 3.4 | 循环小数 | 定义、循环节、简写法 |
| 3.5 | 用计算器探索规律 | 计算器辅助发现模式 |
| 3.6 | 解决问题 | 进一法 vs 去尾法的选择（结合实际场景） |

**核心概念**：
- 循环小数：小数部分，从某一位起，一个或几个数字依次不断重复出现
- 循环节：如 6.3232... 的循环节是 32，简写 6.3̇2̇
- 有限小数 vs 无限小数（循环小数属于无限小数）
- 商的变化规律：
  - 商不变：被除数和除数同扩/同缩相同倍数
  - 除数不变，被除数扩 → 商扩
  - 被除数不变，除数缩 → 商扩

**与第一单元的关系**：小数除法是小数乘法的逆运算，计算方法是对偶的。

### 第五单元：简易方程 ⭐ 全年最核心

**学习目标**：从算术思维跨越到代数思维，会用字母表示数、解方程、列方程解应用题。

| 节次 | 知识点 | 核心内容 |
|------|--------|----------|
| 5.1 | 用字母表示数 | 字母参与运算，乘号可省略：a×a = a² |
| 5.2 | 方程的意义 | 含未知数的等式（必须同时满足：等式 + 有未知数） |
| 5.3 | 等式的性质 | 天平原理：同加减/同乘除（0除外），等式仍成立 |
| 5.4 | 解方程 | 利用等式性质求未知数的值 |
| 5.5 | 实际问题与方程 | 列方程解应用题（设x → 找等量关系 → 列方程 → 解答） |

**核心公式**：
- a² = a×a（注意：不等于 2a = a+a）
- 10个基本数量关系（加/减/乘/除各两个）

**10个数量关系式（列方程的基础）**：
| 运算 | 关系1 | 关系2 |
|------|-------|-------|
| 加法 | 和 = 加数 + 加数 | 加数 = 和 - 另一个加数 |
| 减法 | 差 = 被减数 - 减数 | 减数 = 被减数 - 差 |
| 乘法 | 积 = 因数 × 因数 | 因数 = 积 ÷ 另一个因数 |
| 除法 | 商 = 被除数 ÷ 除数 | 除数 = 被除数 ÷ 商 |

**易错点**：
- 方程 ≠ 等式（方程是特殊的等式，必须含未知数）
- "方程的解"是一个数，"解方程"是一个过程
- 检验习惯：将解代回原方程验证

### 第六单元：多边形的面积

**学习目标**：理解面积公式推导过程（割补、旋转、平移），掌握面积计算。

| 图形 | 面积公式 | 变式（求高/底） |
|------|----------|-----------------|
| 平行四边形 | S = 底×高 = ah | h = S÷a |
| 三角形 | S = 底×高÷2 = ah÷2 | h = S×2÷a |
| 梯形 | S = (上底+下底)×高÷2 = (a+b)h÷2 | h = S×2÷(a+b) |
| 组合图形 | 分割为基本图形后加减 | — |

**公式推导思想（核心方法）**：
- 平行四边形 → 长方形（剪拼、平移）：底=长，高=宽
- 三角形 → 平行四边形（旋转、拼合）：两个完全一样的三角形拼成平行四边形
- 梯形 → 平行四边形（旋转、拼合）：两个完全一样的梯形拼成平行四边形

**重要结论**：
- 等底等高的平行四边形面积相等
- 等底等高的三角形面积相等
- 等底等高的平行四边形面积 = 三角形面积的2倍
- 长方形拉成平行四边形：周长不变，面积变小

**学习目标**：建立数学模型，理解间隔数与棵数的关系。

| 场景 | 公式 | 说明 |
|------|------|------|
| 两端都栽 | 棵数 = 间隔数 + 1 | 路长÷间隔 + 1 |
| 一端栽一端不栽 | 棵数 = 间隔数 | 路长÷间隔 |

**延伸**：鸡兔同笼问题（假设法 + 方程法）
- 假设全是兔 → 先算出鸡的数量
- 假设全是鸡 → 先算出兔的数量
- 方程法：设兔 x 只，列 4x + 2(总头数-x) = 总脚数

### 第二单元：因数与倍数 ⭐ 数论启蒙

**学习目标**：理解因数与倍数的概念，掌握 2/3/5 的倍数特征，认识质数与合数。

| 节次 | 知识点 | 核心内容 |
|------|--------|----------|
| 2.1 | 因数和倍数 | 整除条件下，大数是小数的倍数，小数是大数的因数 |
| 2.2 | 2和5的倍数特征 | 2的倍数：个位 0/2/4/6/8（偶数）；5的倍数：个位 0/5 |
| 2.3 | 3的倍数特征 | 各位数字之和是3的倍数 |
| 2.4 | 质数和合数 | 质数：只有2个因数；合数：至少3个因数；1既不是质数也不是合数 |
| 2.5 | 奇偶性探究 | 奇±偶=奇，奇±奇=偶，偶±偶=偶 |

**知识结构**：
```
自然数按能否被2整除
├── 奇数（个位1/3/5/7/9）
└── 偶数（个位0/2/4/6/8，0也是偶数）

自然数按因数个数
├── 1（只有1个因数）
├── 质数（恰好2个因数：1和自身）
└── 合数（3个及以上因数）
```

**必记数据**：
- 20以内质数（8个）：2, 3, 5, 7, 11, 13, 17, 19
- 100以内质数（25个）
- 最小的质数 = 2（唯一的偶质数）
- 最小的合数 = 4
- 完全数：6, 28...

**进阶概念**（为分数约分/通分打基础）：
- 互质数：公因数只有1的两个数
- 最大公因数（短除法）
- 最小公倍数（短除法）
- 分解质因数（短除法）

### 第三单元：长方体和正方体

**学习目标**：掌握长方体和正方体的特征、表面积和体积的计算。

#### 3.1 特征对比

| 特征 | 长方体 | 正方体 |
|------|--------|--------|
| 面 | 6个面（长方形，最多2个正方形） | 6个面（全等正方形） |
| 棱 | 12条（相对的棱相等） | 12条（全部相等） |
| 顶点 | 8个 | 8个 |
| 关系 | — | 特殊的长方体 |

#### 3.2 棱长

| 公式 | 表达 |
|------|------|
| 长方体棱长总和 | L = (长+宽+高)×4 |
| 正方体棱长总和 | L = 棱长×12 |

#### 3.3 表面积

| 场景 | 公式 |
|------|------|
| 完整6个面 | S = (长×宽 + 长×高 + 宽×高)×2 |
| 无盖/无底（5个面） | S = 长×宽 + (长×高 + 宽×高)×2 |
| 无底无盖（4个面） | S = (长×高 + 宽×高)×2 |
| 正方体 | S = 棱长×棱长×6 |

**生活场景**：油箱(6面)、鱼缸(5面)、烟囱(4面)

#### 3.4 体积

| 公式 | 表达 |
|------|------|
| 长方体体积 | V = 长×宽×高 = abh |
| 正方体体积 | V = 棱长³ = a³ |
| 通用公式 | V = 底面积×高 = Sh |

#### 3.5 体积单位换算

```
1立方米 = 1000立方分米 = 1000000立方厘米
1立方分米 = 1升(L) = 1000立方厘米 = 1000毫升(ml)
1立方厘米 = 1毫升(ml)
```

#### 3.6 不规则物体体积（排水法）

- V物体 = V现在 - V原来
- V物体 = 底面积 × h升高

**重要结论**：
- 长/宽/高同时扩大n倍 → 表面积扩大n²倍，体积扩大n³倍
- 截成若干块 → 表面积增加，体积不变

### 第四单元：分数的意义和性质 ⭐ 全年最核心

**学习目标**：系统建立分数概念体系，为分数运算打基础。

#### 4.1 分数的意义

| 概念 | 定义 |
|------|------|
| 单位"1" | 一个整体（可以是物体、群体等） |
| 分数 | 把单位"1"平均分成若干份，表示其中一份或几份 |
| 分数单位 | 分子为1的分数（如 4/5 的分数单位是 1/5） |

#### 4.2 分数与除法

**A÷B = A/B**（B≠0）

#### 4.3 真分数·假分数·带分数

| 类型 | 条件 | 与1的关系 |
|------|------|-----------|
| 真分数 | 分子 < 分母 | < 1 |
| 假分数 | 分子 ≥ 分母 | ≥ 1 |
| 带分数 | 整数 + 真分数 | > 1 |

**互化方法**：
- 假分数 → 带分数：分子÷分母，商为整数部分，余数为新分子
- 带分数 → 假分数：整数×分母+分子 = 新分子，分母不变
- 整数 → 假分数：整数×分母 = 新分子

#### 4.4 分数的基本性质

> 分数的分子和分母同时乘以或除以相同的数（0除外），分数的大小不变。

**（类比：商不变性质，本质相同）**

#### 4.5 约分和通分

| 操作 | 方法 | 目的 |
|------|------|------|
| 约分 | 分子分母同时÷公因数 | 化为最简分数 |
| 通分 | 异分母 → 同分母（取最小公倍数做公分母） | 便于比较和加减 |
| 最简分数 | 分子分母只有公因数1 | 最终化简形式 |

#### 4.6 分数和小数互化

| 方向 | 方法 |
|------|------|
| 小数→分数 | 数位数定分母（0.3=3/10, 0.03=3/100） |
| 分数→小数 | 分子÷分母，或化为10/100/1000为分母 |

**常用分数小数对照**：

| 分数 | 小数 | 分数 | 小数 |
|------|------|------|------|
| 1/2 | 0.5 | 1/4 | 0.25 |
| 3/4 | 0.75 | 1/5 | 0.2 |
| 2/5 | 0.4 | 3/5 | 0.6 |
| 4/5 | 0.8 | 1/8 | 0.125 |
| 1/20 | 0.05 | 1/25 | 0.04 |

**学习目标**：用天平找次品，理解最优策略（三分法）。

**核心策略**：把物品尽可能平均分成3份。

| 物品数量 | 最少称量次数 |
|----------|-------------|
| 2~3 | 1次 |
| 4~9 | 2次 |
| 10~27 | 3次 |
| 28~81 | 4次 |
| 82~243 | 5次 |
| 244~729 | 6次 |

**规律**：称n次最多可分辨 3ⁿ 个物品。

## 跨单元知识串联

### 串联1："转化思想"贯穿几何
- 上册U6 多边形面积：平行四边形→长方形，三角形→平行四边形，梯形→平行四边形
- 下册U3 长方体正方体：不规则物体体积 → 排水法转化
- **核心**：把未知图形转化为已知图形来计算

### 串联2：从算术到代数
- 上册U1~U3 小数运算（算术思维）
- 上册U5 简易方程（代数思维入门）
- **跨越点**：从"列式计算"到"设未知数找等量关系"

### 串联3：数论为分数铺路
- 下册U2 因数与倍数 → 公因数/公倍数 → 约分/通分
- 下册U4 分数基本性质 → 约分通分的理论依据
- 下册U6 分数加减 → 约分通分的实际应用
- **链条**：因数倍数 → 分数性质 → 分数运算

### 串联4：从平面到立体
- 上册U6 多边形面积（二维）
- 下册U1 观察物体（二维与三维的桥梁）
- 下册U3 长方体正方体（三维）
- **发展路径**：面积→空间观念→体积

## 备考建议

### 上册期中（U1~U4）重点
1. 小数乘除法计算（必须练到又快又准）
2. 运算定律的灵活运用（尤其是分配律）
3. 循环小数的判断和表示
4. 可能性的计算

### 上册期末（U5~U7）重点
1. 解方程（各种类型：一步、两步）
2. 列方程解应用题（找等量关系是关键）
3. 多边形面积（含组合图形）
4. 植树问题的三种情况

### 下册期中（U1~U3）重点
1. 因数和倍数的基本概念
2. 2/3/5的倍数特征
3. 质数和合数的区分
4. 长方体正方体表面积和体积计算
5. 体积单位换算

### 下册期末（U4~U8）重点
1. 分数的意义和性质（真假分数、约分通分）
2. 分数与小数互化
3. 分数加减运算（含混合运算和简便运算）
4. 折线统计图的绘制和分析
