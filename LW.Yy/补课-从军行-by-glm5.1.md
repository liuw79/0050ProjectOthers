# 补课材料：从军行

> **出处**：部编版五年级下册 · 第9课《古诗三首》
> **创建时间**：2026-03-30
> **生成模型**：GLM-5.1

---

## 原文（背诵）

> **从军行**
> （唐）王昌龄
>
> 青海长云暗雪山，
> 孤城遥望玉门关。
> 黄沙百战穿金甲，
> 不破楼兰终不还。

---

## 逐句翻译

| 原句 | 译文 |
|------|------|
| 青海长云暗雪山 | 青海湖上空，层层云朵遮暗了连绵的雪山 |
| 孤城遥望玉门关 | 戍守在孤城中的将士，远远望着玉门关 |
| 黄沙百战穿金甲 | 在漫天黄沙中身经百战，铁甲都被磨穿了 |
| 不破楼兰终不还 | 不打败进犯的敌人，决不返回家乡 |

---

## 必背知识点

### 1. 作者

**王昌龄**（唐代）—— "七绝圣手"，擅长边塞诗。

### 2. 重点字词（考试爱考）

| 字词 | 释义 |
|------|------|
| **青海** | 青海湖（不是青海省） |
| **暗** | 使……暗淡（遮蔽） |
| **孤城** | 玉门关外的孤城 |
| **穿** | **磨穿、磨破**（不是"穿戴"） |
| **楼兰** | 西域古国名，这里**泛指敌人** |
| **从军行** | 乐府旧题，多写军旅生活 |

### 3. 主旨

描写西北边塞的艰苦环境，表现戍边将士**保家卫国、誓死破敌**的英雄气概。

### 4. 情感基调

**悲壮豪迈** —— 环境苦、战事苦，但斗志不灭。

---

## 前两句 vs 后两句

| | 内容 | 作用 |
|--|------|------|
| 前两句 | 青海长云暗雪山，孤城遥望玉门关 | 写**景**：边塞环境的壮阔苍凉 |
| 后两句 | 黄沙百战穿金甲，不破楼兰终不还 | 抒**情**：将士的坚定决心 |

> 技巧：先渲染环境的艰苦，反衬出将士意志的坚强。

## 背诵技巧

第1步：画面记忆法（脑海中构建画面）
```
青海湖 → 云层遮住雪山 → 远处一座孤城 → 看到玉门关
→ 黄沙漫天 → 将士铠甲磨破 → 但眼神坚定 → "不破楼兰终不还！"
```

第2步：关键字串联
```
青海 → 长云 → 雪山 → 孤城 → 玉门关
黄沙 → 百战 → 金甲 → 楼兰 → 终不还
```

## 常考题型

### 题型1：解释词语
> "黄沙百战穿金甲"中的"穿"是什么意思？
> **答**：磨穿、磨破。说明战斗频繁、时间长久。

### 题型2：理解诗句
> "不破楼兰终不还"表达了什么？
> **答**：表达了戍边将士保家卫国、誓死打败敌人的坚定决心和英雄气概。

### 题型3：情感对比（和同课另外两首一起考）
> 三首诗都表达爱国情怀，但情感有何不同？
> - 《从军行》→ **豪迈坚定**
> - 《秋夜将晓……》→ **悲愤忧虑**
> - 《闻官军……》→ **喜悦激动**
