### 4. 圈子经营奖金（按季度核发）

#### 4.1 华南/华东圈

```
奖金基数 = 圈子经营利润 × 3%
         = (收入 - 运营成本) × 3%  // 不减人员成本
```

**开启条件：**
- 必达成：季度会员消卡率60%（与上新数双向锁定）
- 叠加条件1：季度产生后端订单会员企业占比≥30% → 基数×1.2
- 叠加条件2：年度整体业绩增长≥10% → 基数×1.2

**分配原则：**
- 圈长：个人业务奖金×60% + 经营奖金包×60%
- 副圈长：个人业务奖金×90% + 经营奖金包×30%
- 非业务伙伴：经营奖金包×10%

#### 4.2 华北圈（阶梯制）

| GMV增长率 | 经营奖金包 |
|----------|-----------|
| ≥30% | 12万 |
| ≥60% | 24万 |
| ≥90% | 48万 |
| ≥120% | 72万 |

**开启条件：** 月均开课2门 + 圈子经营净利润为正

**分配原则：**
- 圈长：个人业务奖金×60% + 经营奖金包×60%
- 副圈长：经营奖金包×30%（26年暂不设，由圈长兼任）
- 团队其他：经营奖金包×10%

## 七、测算假设
