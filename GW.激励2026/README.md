# 2026年激励方案测算工具

针对高维2026年激励方案的整体测算工具。

## 功能概述

- **分经营圈测算**：华南/华东/华北/KA圈的奖金占收入占比
- **分课题圈测算**：三大课题圈的奖金占公开课收入占比
- **全公司维度测算**：两种收入预测版本（持平/增长）
- **多场景对比**：保守/中性/乐观三种场景

## 测算版本说明

| 版本 | 说明 |
|------|------|
| 版本一 | 收入与25年持平 |
| 版本二 | KA圈+50%, 华南华东+10%, 华北+50% |

## 场景假设

| 场景 | 叠加条件 | 华北GMV增长 | 超预期上新 | 关键战役奖金 |

## 使用步骤

1. 将财务数据填入 `data/input_data.csv`
2. 运行 `python src/run_calculator.py`
3. 选择预测版本和测算场景
4. 查看生成的报告 `docs/测算报告-*.md`
