# 新场地快速添加指南

## 📋 添加新场地的标准流程

### 第一步：复制模板
1. 复制 `场地数据/场地016_新场地模板/` 整个文件夹
2. 重命名为 `场地0XX_[场地名称]/`（XX为下一个编号）
3. 进入文件夹，编辑 `场地信息.md`

### 第二步：基本信息录入
**必填项目**：
- 场地编号（按顺序递增）
- 楼盘名称
- 楼层
- 详细地址
- 记录日期
- 记录人

### 第三步：关键数据测量
**空间规格**（影响30分）：
- 单间净宽（柱子到墙距离）
- 单间净高（板到板高度）
- 总建筑面积

**费用信息**（影响30分）：
- 租金单价（元/㎡/月）
- 物业管理费
- 空调计费方式
- 年总费用计算

**基础设施**（影响20分）：
- 女洗手间数量
- 所在区域（南山/福田）
- 露台/阳台情况

### 第四步：量化评分
**使用评分标准**：

| 项目 | 评分标准 |
|------|----------|
| **课室宽度** | ≥10m=15分，8-10m=11.25分，7.5-8m=7.5分，<7.5m=0分 |
| **层高** | ≥4.2m=15分，4.0-4.2m=13.5分，3.8-4.0m=11.25分，3.5-3.8m=7.5分，<3.5m=0分 |
| **年费用** | ≤100万=15分，100-110万=13.5分，110-120万=11.25分，120-130万=7.5分，130-140万=4.5分，140-150万=3分，150-160万=2.25分，160-170万=1.5分，>170万=0分 |
| **空调计费** | 按流量=15分，其他=0分 |
| **总面积** | 800-1000㎡=10分，1000-1200㎡=6分，其他=0分 |
| **区域位置** | 南山/福田=5分，其他=0分 |
| **女洗手间** | ≥6个=5分，3-5个=2.5分，<3个=0分 |
| **露台** | 有=5分，无=0分 |
| **装修状态** | 现成课室=5分，小改=2.5分，毛坯=0分 |
| **周边环境** | 很好=5分，一般=3分，稍差=2分，较差=0分 |
| **配套设施** | 齐全=5分，较好=4分，一般=3分，较差=2分，很差=0分 |

### 第五步：文档同步更新
**必须更新的文档**：

1. **场地对比分析.md**
   - 候选场地基本信息表格
   - 详细评分对比表格
   - 按层高排序表格
   - 费用对比分析表格
   - 空间规格对比表格

2. **progress_2025-07-11.md**
   - 新增场地工作记录
   - 排名变化说明
   - 关键决策依据

### 第六步：质量检查
**使用检查清单**：

## 🚀 快速操作命令

### 创建新场地文件夹
```bash
# 复制模板（在GW.Classroom目录下执行）
cp -r "场地数据/场地016_新场地模板" "场地数据/场地017_[新场地名称]"

# 进入新文件夹
cd "场地数据/场地017_[新场地名称]"

# 编辑场地信息
open 场地信息.md
```

### 批量替换模板内容
在场地信息.md中，使用查找替换功能：
- `016` → `017`（新编号）
- `[场地名称]` → `实际场地名称`
- `[楼盘名称]` → `实际楼盘名称`
- 其他占位符按实际情况替换

## ⚠️ 常见错误预防

### 评分计算错误
- **费用评分**：严格按照区间计算，不要四舍五入
- **空间评分**：以实测数据为准，不要估算
- **总分计算**：使用Excel公式验证

### 数据不一致
- **露台信息**：确保评分与描述一致
- **装修状态**：统一标准（现成课室/小改/毛坯）
- **费用数据**：各表格中的费用信息必须一致

### 文档同步遗漏
- **检查所有表格**：确保新场地出现在所有相关表格中
- **排名更新**：新场地加入后重新排序
- **状态标记**：正确标记查看状态和决策状态

## 📊 Excel辅助工具

### 评分计算公式
```excel
# 综合得分
=ROUND(AVERAGE(B2:L2),1)

# 费用评分（假设年费用在M2）
=IF(M2<=100,15,IF(M2<=110,13.5,IF(M2<=120,11.25,IF(M2<=130,7.5,IF(M2<=140,4.5,IF(M2<=150,3,IF(M2<=160,2.25,IF(M2<=170,1.5,0))))))))

# 排名计算
=RANK(总分列,$总分列$2:$总分列$20,0)
```

### 数据验证
```excel
# 评分范围检查
=IF(AND(B2>=0,B2<=15),"✓","❌评分超出范围")

# 完整性检查
=IF(COUNTA(B2:L2)=11,"✓完整","⚠️缺失"&(11-COUNTA(B2:L2))&"项")
```
