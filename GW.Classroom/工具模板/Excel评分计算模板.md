# Excel评分计算模板

## 📊 标准评分公式

### 基础设置

**假设数据布局**：
- A列：场地名称
- B-L列：11项评分指标（地理位置、交通便利性、空间布局等）
- M列：综合得分
- N列：排名

### 🧮 核心计算公式

#### 1. 综合得分计算
```excel
=ROUND(AVERAGE(B2:L2),1)
```
**说明**：计算11项指标的平均分，保留1位小数

#### 2. 加权综合得分（如需要）
```excel
=ROUND((B2*0.15+C2*0.15+D2*0.1+E2*0.1+F2*0.1+G2*0.1+H2*0.1+I2*0.05+J2*0.05+K2*0.05+L2*0.05),1)
```
**权重分配**：
- 地理位置：15%
- 交通便利性：15%
- 空间布局：10%
- 装修状况：10%
- 配套设施：10%
- 租金水平：10%
- 管理服务：10%
- 发展潜力：5%
- 周边环境：5%
- 安全性：5%
- 灵活性：5%

#### 3. 排名计算
```excel
=RANK(M2,$M$2:$M$14,0)
```
**说明**：根据综合得分进行降序排名

### 📈 费用分析公式

#### 1. 月租金总额
```excel
=面积*单价租金
```

#### 2. 年度总成本
```excel
=(月租金+月管理费+月空调费)*12+年停车费
```

#### 3. 性价比指数
```excel
=ROUND(综合得分/(年度总成本/10000),2)
```
**说明**：得分除以万元成本，数值越高性价比越好

### 🎯 条件格式设置

#### 1. 评分等级颜色
```excel
条件1：>=8.0  背景色：绿色（优秀）
条件2：>=7.0  背景色：黄色（良好）
条件3：>=6.0  背景色：橙色（一般）
条件4：<6.0   背景色：红色（较差）
```

#### 2. 排名突出显示
```excel
条件1：排名=1   背景色：金色
条件2：排名<=3  背景色：银色
条件3：排名<=5  背景色：铜色
```

### 📊 数据验证规则

#### 1. 评分范围限制
```excel
数据验证：整数，1-10之间
错误提示："评分必须在1-10之间"
```

#### 2. 必填字段检查
```excel
=IF(ISBLANK(B2),"缺少地理位置评分","")
```

### 🔧 实用辅助公式

#### 1. 完整性检查
```excel
=IF(COUNTA(B2:L2)=11,"✓完整","⚠️缺失"&(11-COUNTA(B2:L2))&"项")
```

#### 2. 评分统计
```excel
最高分：=MAX(B2:L2)
最低分：=MIN(B2:L2)
平均分：=AVERAGE(B2:L2)
标准差：=STDEV(B2:L2)
```

#### 3. 同分处理
```excel
=IF(COUNTIF($M$2:$M$14,M2)>1,"并列第"&RANK(M2,$M$2:$M$14,0)&"名","第"&RANK(M2,$M$2:$M$14,0)&"名")
```

### 📋 快速对比公式

#### 1. 两场地对比
```excel
=IF(场地A得分>场地B得分,"场地A更优",IF(场地A得分<场地B得分,"场地B更优","得分相同"))
```

#### 2. 优势项目识别
```excel
=INDEX({"地理位置";"交通便利性";"空间布局";"装修状况";"配套设施";"租金水平";"管理服务";"发展潜力";"周边环境";"安全性";"灵活性"},MATCH(MAX(B2:L2),B2:L2,0))
```

#### 3. 劣势项目识别
```excel
=INDEX({"地理位置";"交通便利性";"空间布局";"装修状况";"配套设施";"租金水平";"管理服务";"发展潜力";"周边环境";"安全性";"灵活性"},MATCH(MIN(B2:L2),B2:L2,0))
```
