# 课室场地筛选项目

## 项目简介
本项目旨在为深圳南山/福田区域的3间课室场地筛选建立一套完整的信息收集和决策模型系统。

## 项目结构
```
GW.Classroom/
├── progress.md              # 项目进度文档
├── 场地筛选决策模型.md      # 决策模型和评分体系
├── 场地信息记录表.md        # 标准化信息记录模板
├── README.md               # 项目说明文档
├── 场地数据/               # 场地信息存储目录
│   ├── 场地001/           # 具体场地文件夹
│   ├── 场地002/
│   └── ...
├── 工具模板/              # Excel等工具模板
└── 文档/                  # 其他相关文档
```

## 核心功能
1. **基本要求筛选**: 确保所有场地满足基本要求
2. **量化评分系统**: 11个维度的100分制评分体系
3. **标准化记录**: 统一的信息采集和记录格式
4. **决策支持**: 基于数据的科学决策流程

## 快速开始
1. 查看 `progress.md` 了解当前项目状态
2. 使用 `场地信息记录表.md` 模板记录场地信息
3. 参考 `场地筛选决策模型.md` 进行评分和决策

## 关键指标
- **预算**: 100-120万/年
- **面积**: 800-1000平米
- **课室**: 3间，单间10m×18m×≥4m
- **区域**: 南山/福田优先

## 注意事项
- 所有数据必须现场核实
- 保持文档结构清晰
- 避免复杂化，专注核心需求
