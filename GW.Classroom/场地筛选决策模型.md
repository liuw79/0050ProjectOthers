# 课室场地筛选决策模型

## 一、项目基本要求

### 1.1 空间规格要求
- **课室数量**: 3间独立课室
- **单间尺寸**: 净宽10m × 净长18m × 净高≥4m
- **内部结构**: 方正，中间无柱子遮挡
- **总面积**: 800-1000平米（含公共区域）

### 1.2 费用约束
- **年总费用**: 100-120万元（理想范围）
- **费用构成**: 租金 + 物业管理费 + 空调费 + 停车费 + 其他

### 1.3 基础设施要求
- **女洗手间**: ≥6个蹲位/坐厕
- **空调系统**: 按流量计费（非包干制）
- **建筑质量**: 写字楼竣工≤15年或重装≤5年

### 1.4 区位要求
- **区域**: 优先南山、福田
- **周边环境**: 200m内无大型工地、无城中村

## 三、决策流程

### 3.1 筛选流程
1. **基本信息收集** → 使用标准化记录表
2. **初步筛选** → 排除明显不符合的场地
3. **实地勘察** → 详细测量和评估
4. **量化评分** → 按照评分体系打分
5. **排序对比** → 生成综合排序
6. **最终决策** → 结合预算和实际需求

### 3.2 决策权重说明
- **空间规格（30分）**: 课室宽度+层高，使用体验的核心
- **费用控制（30分）**: 年费用+空调计费，成本控制的关键
- **基础配套（20分）**: 面积+地理位置+洗手间，基本需求
- **舒适度（20分）**: 露台+装修+环境+配套，体验提升

### 3.3 决策建议框架
- **80分以上**: 优先推荐
- **70-80分**: 重点考虑
- **60-70分**: 备选方案
- **60分以下**: 不推荐

## 四、风险评估

### 4.1 主要风险
1. **预算风险**: 市场价格普遍超出预算
2. **空间风险**: 标准规格场地稀缺
3. **时间风险**: 好场地竞争激烈
4. **装修风险**: 改造成本和周期不确定

### 4.2 风险控制
- 设定预算弹性区间
- 准备备选方案
- 建立快速决策机制
- 提前评估改造成本

## 五、使用说明

### 5.1 评分步骤
1. 收集场地基本信息
2. 实地测量关键指标
3. 按评分标准逐项打分
4. 计算总分并排序
5. 结合实际情况做最终决策

### 5.2 注意事项
- 所有数据必须现场核实
- 重点关注费用和空间的平衡
- 考虑装修改造的时间成本
- 保持决策灵活性，避免过度僵化
