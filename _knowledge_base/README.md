# 知识库文件夹

这个文件夹存放调研资料和知识整理。

## 文件命名规范

格式：`主题-YYYY-MM-DD.md`

示例：
- `Claude-Code-2025-01-09.md`
- `AI写作工具对比-2025-01-09.md`

## 文件内容规范

每个知识库文件应包含：

```markdown
# [主题] - 知识库

## 信息收集时间
YYYY年MM月DD日

## 信息来源
- 官方文档: [URL]
- 科技媒体: [URL]
- 社区讨论: [URL]
- 其他: [URL]

## 关键发现

### 基本信息
[产品/技术的基本信息]

### 核心特性
1. 特性1
2. 特性2
3. ...

### 与竞品对比
[对比分析]

### 用户反馈
[社区、论坛的用户反馈]

### 注意事项
[需要注意的问题、限制等]

## 数据和统计
[如有]

## 下次更新建议
建议每[周/月/季度]更新一次，关注[具体方面]

## 相关资源
- [链接]
```

---

## 维护建议

- 每次写作前，检查是否有相关的知识库文件
- 如果信息过时（超过1个月），重新搜索更新
- 定期清理不再需要的文件

---

最后更新：[日期]
