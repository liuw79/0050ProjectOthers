## 产品文档：CEO人才雷达 - MVP版

### 1. 引言

*   **1.1 产品概述**
    *   CEO人才雷达是一款专为创业公司CEO设计的轻量级关键人才追踪小程序。它旨在帮助CEO高效记录、管理和跟进潜在的核心人才，确保不错过任何重要的招募机会。MVP版本聚焦于最核心的记录和提醒功能。
*   **1.2 目标用户**
    *   创业公司CEO及创始人。
*   **1.3 核心价值**
    *   **便捷记录：** 随时随地记录潜在人才信息。
    *   **有效提醒：** 不会忘记跟进重要人才。
    *   **初步管理：** 形成个人专属的初步人才备忘录。

### 2. MVP功能规格

*   **2.1 核心人才信息录入**
    *   **描述：** 用户可以手动输入人才的关键信息。
    *   **字段：**
        *   姓名 (必填)
        *   联系方式 (如电话/微信，选填)
        *   来源渠道 (如：朋友推荐、行业会议、社交媒体等，选填)
        *   初步备注 (文本输入，选填，用于记录第一印象、关键技能等)
        *   重要性标签 (单选：非常重要 / 重点跟进 / 一般关注)
*   **2.2 手动设置互动提醒**
    *   **描述：** 用户可以为已录入的每个人才设置下一次需要互动的日期和简单的提醒事项。
    *   **功能：**
        *   选择提醒日期。
        *   输入提醒内容 (如：约咖啡、发送公司进展)。
        *   小程序在指定日期推送系统通知 (依赖用户开启小程序通知权限)。
*   **2.3 互动记录**
    *   **描述：** 用户可以为每个人才添加简单的互动记录。
    *   **字段：**
        *   互动日期 (自动记录或可修改)
        *   互动类型 (可选：微信沟通 / 电话 / 约见 / 其他)
        *   简要内容 (文本输入)
*   **2.4 基础人才列表与查看**
    *   **描述：** 展示已录入的人才列表，并可以查看详情。
    *   **列表展示信息：** 姓名、重要性标签、下次提醒日期（如有）。
    *   **排序：** 默认按添加时间倒序，可选按下次提醒日期正序。
    *   **筛选：** 可按重要性标签筛选。
    *   **详情页：** 展示人才所有录入信息及互动记录列表。

### 3. MVP UI/UX规划

保持“极轻”原则，界面简洁直观，操作便捷。

*   **3.1 主要页面**

    1.  **人才列表页 (首页)**
    2.  **添加/编辑人才页**
    3.  **人才详情页**
    4.  **添加互动记录页**

*   **3.2 页面详细设计**

    **3.2.1 人才列表页 (首页)**
    *   **顶部：**
        *   搜索框 (MVP阶段可简化为按姓名模糊搜索，或暂时不做，依赖筛选)。
        *   筛选按钮 (按重要性标签筛选)。
    *   **中间：**
        *   人才卡片列表。每个卡片显示：
            *   姓名
            *   重要性标签 (用不同颜色或图标区分)
            *   下次提醒日期和事项 (若有，简洁展示)
        *   若无人才，显示友好提示和“添加人才”引导。
    *   **底部：**
        *   “+ 添加人才”悬浮按钮。

    **用户流程：**
    1.  打开小程序进入此页面。
    2.  可浏览人才列表。
    3.  可点击筛选按钮进行筛选。
    4.  可点击某个人才卡片进入人才详情页。
    5.  可点击“+ 添加人才”按钮进入添加人才页。

    **示意图 (文字描述):**
    ```
    +--------------------------------------+
    | [搜索框/筛选按钮]                    |
    +--------------------------------------+
    | [人才卡片1]                          |
    |   张三     [非常重要]                |
    |   提醒: 明天 约咖啡                  |
    +--------------------------------------+
    | [人才卡片2]                          |
    |   李四     [重点跟进]                |
    |                                      |
    +--------------------------------------+
    | ...                                  |
    +--------------------------------------+
    |                                (+)   |  <-- 添加人才悬浮按钮
    +--------------------------------------+
    ```

    **3.2.2 添加/编辑人才页**
    *   **表单项：**
        *   姓名 (输入框)
        *   联系方式 (输入框)
        *   来源渠道 (输入框或简单选择器)
        *   初步备注 (多行文本输入框)
        *   重要性标签 (单选按钮组：非常重要 / 重点跟进 / 一般关注)
        *   下次提醒日期 (日期选择器)
        *   提醒事项 (输入框)
    *   **底部：**
        *   “保存”按钮。
        *   (编辑时) “删除人才”按钮。

    **用户流程：**
    1.  从人才列表页点击“+ 添加人才”进入。
    2.  填写表单信息。
    3.  点击“保存”。

    **示意图 (文字描述):**
    ```
    +--------------------------------------+
    | 添加新人才                           |
    +--------------------------------------+
    | 姓名: [___________]                  |
    | 联系方式: [___________]              |
    | 来源: [___________]                  |
    | 备注: [_________________________]    |
    | 重要性: ( )非常重要 ( )重点 ( )一般  |
    |                                      |
    | -- 互动提醒 (可选) --                |
    | 提醒日期: [选择日期]                 |
    | 提醒事项: [___________]              |
    +--------------------------------------+
    |              [ 保存 ]                |
    +--------------------------------------+
    ```

    **3.2.3 人才详情页**
    *   **顶部：**
        *   人才姓名。
        *   “编辑”按钮 (跳转到编辑人才页)。
    *   **信息展示区：**
        *   联系方式、来源渠道、初步备注、重要性标签。
        *   下次提醒日期和事项。
    *   **互动记录区：**
        *   “+ 添加互动记录”按钮。
        *   互动记录列表 (按时间倒序，显示日期、类型、内容摘要)。

    **用户流程：**
    1.  从人才列表页点击人才卡片进入。
    2.  查看人才详细信息和互动历史。
    3.  可点击“编辑”修改人才信息。
    4.  可点击“+ 添加互动记录”添加新的互动。

    **示意图 (文字描述):**
    ```
    +--------------------------------------+
    | 张三                     [编辑]      |
    +--------------------------------------+
    | 联系: 138********                    |
    | 来源: 朋友推荐                       |
    | 备注: 技术大牛，关注中               |
    | 重要性: 非常重要                     |
    | 提醒: 明天 约咖啡                    |
    +--------------------------------------+
    | 互动记录                [+ 添加互动] |
    +--------------------------------------+
    | - 2023-10-26 微信沟通: 初步了解      |
    | - 2023-10-20 电话: 确认意向          |
    | ...                                  |
    +--------------------------------------+
    ```

    **3.2.4 添加互动记录页**
    *   **表单项：**
        *   互动日期 (日期选择器，默认为当天)
        *   互动类型 (选择器：微信沟通 / 电话 / 约见 / 其他)
        *   互动内容 (多行文本输入框)
    *   **底部：**
        *   “保存记录”按钮。

    **用户流程：**
    1.  从人才详情页点击“+ 添加互动记录”进入。
    2.  填写互动信息。
    3.  点击“保存记录”。

    **示意图 (文字描述):**
    ```
    +--------------------------------------+
    | 添加互动记录 (张三)                  |
    +--------------------------------------+
    | 互动日期: [选择日期]                 |
    | 互动类型: [选择类型 ▼]               |
    | 互动内容: [_________________________]    |
    |                                      |
    +--------------------------------------+
    |           [ 保存记录 ]               |
    +--------------------------------------+
    ```

*   **3.3 通知**
    *   当设置的互动提醒到期时，小程序应能通过微信的服务通知或订阅消息（需用户授权）推送提醒给用户。
    *   提醒内容应包含人才姓名和提醒事项。

### 4. 非功能性需求 (MVP)
*   **易用性：** 界面简洁，操作流程符合用户习惯。
*   **性能：** 列表加载、页面跳转流畅。
*   **数据存储：** 用户数据存储在小程序云数据库或自建服务器，确保基本安全。

这份文档和UI规划为您MVP阶段的开发提供了基础框架。在实际开发中，可以根据微信小程序的UI规范和组件库进行细化。
