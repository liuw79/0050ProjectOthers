# wePark 停车记录小程序

## 简介

wePark 是一款简洁实用的微信小程序，旨在帮助用户快速记录停车位置和时间，并支持拍照记录，方便查找车辆。

## 主要功能

- **楼层选择**: 提供多种楼层按钮（大按钮和小按钮布局），方便用户快速选择停车楼层。
- **自动记录时间**: 选择楼层后，自动记录当前时间，并格式化显示（年-月-日 时:分）。
- **拍照记录**: 
    - 默认开启相机功能（可在首页开关）。
    - 选择楼层后自动进行拍照。
    - 照片与停车记录关联，方便后续查看。
- **历史记录**: 
    - 在"历史"页面查看所有停车记录列表。
    - 列表显示楼层、格式化的停车时间，并标注"今天"、"昨天"或"前天"。
    - 显示与记录关联的照片缩略图。
    - 点击缩略图可查看照片大图。
- **数据管理**: 
    - 历史记录页面提供"导出"功能，可将记录复制到剪贴板（CSV格式）。
    - 提供"清空"功能，可删除所有历史记录。
- **数据持久化**: 用户的停车记录（包括照片信息）和相机开关设置会保存在本地存储中。
- **分享功能**: 
    - 支持小程序右上角转发。
    - 首页分享会包含当前停车信息。
    - 历史页分享会包含最近一条停车信息。

## 页面说明

- **停车页面 (首页)**: 
    - 提供楼层选择按钮。
    - 默认开启相机，用户可手动切换开关。
    - 选择楼层后自动记录时间并拍照（若相机开启）。
    - 底部显示当前记录的停车时间和日期文本。
    - 支持分享当前停车信息。
- **历史页面**: 
    - 展示所有历史停车记录列表。
    - 提供照片预览、导出记录和清空记录的功能。
    - 支持分享最近的停车记录。

## 使用说明

1. 在微信开发者工具中导入项目。
2. 在 `project.config.json` 文件中填入您的小程序 AppID。
3. 编译并运行小程序。
4. 在首页选择停车楼层，应用会自动记录时间和拍照。
5. 前往"历史"页面查看、管理或分享记录。

## 注意事项

- 请确保已授权小程序的相机权限，以便正常使用拍照功能。
- 分享功能需要确保 `/images/share.png` 文件存在。
- 导出功能将数据复制到剪贴板，请自行粘贴保存。
