# 个人素材库 - 使用说明

这是降AI味的核心！这个文件夹存放你的真实经历、观点、案例。

## 为什么需要个人素材库？

AI写作最大的问题是"AI腔"——空洞、套话、没有真实感。

解决方法：用你的真实经历和观点替代AI的编造。

**示例对比：**

❌ AI腔：
> "Claude Code显著提升了开发效率，让编程工作变得更加高效便捷。"

✅ 用真实素材改写：
> "我用Claude Code重构了3个项目，平均每个省了5小时。特别是处理重复代码，之前要半天，现在20分钟搞定。"

差别在哪？**真实的数字、真实的场景、真实的感受。**

## 推荐的素材来源

### 1. 社交媒体动态（强烈推荐）

如果你在即刻、Twitter、朋友圈等平台发过动态，那些就是最好的素材！

**为什么？**
- 真实的想法（当时真实的吐槽和思考）
- 真实的语言（你平时怎么说话）
- 真实的经历（亲身经历的事）

**如何整理：**
1. 导出你的动态（如即刻支持导出）
2. 保存为CSV或文本文件
3. 放入这个文件夹

**文件命名示例：**
- `全部即刻动态.csv`
- `Twitter存档.txt`
- `朋友圈精选.md`

### 2. 工作笔记/日记

你的工作笔记、项目总结、每日思考等。

**文件命名示例：**
- `工作笔记-2024.md`
- `项目复盘.md`
- `每日思考.md`

### 3. 已发布的文章/帖子

你在其他平台发布的内容（知乎回答、论坛帖子等）。

**文件命名示例：**
- `知乎回答精选.md`
- `论坛帖子存档.md`

### 4. 主题分类素材（可选）

如果你经常写某些主题，可以创建主题文件。

**文件命名示例：**
- `AI工具使用经历.md`
- `产品开发心得.md`
- `创业踩坑记录.md`

## AI如何使用你的素材？

### 使用场景1: 文章开头（真实经历引入）

你的素材：
> "用了Claude Code两周，最大感受：终于不用自己写重复代码了。"

AI会改写成：
> "两周前开始用Claude Code。第一感觉？不用自己写重复代码了，这个太爽了。"

### 使用场景2: 观点支撑（真实评价增强可信度）

你的素材：
> "发现Claude Code的一个问题：处理大文件时会卡顿。"

AI会改写成：
> "但Claude Code也不是完美的。我测试发现，处理超过1000行的代码文件时，响应时间会明显变长。"

### 使用场景3: 案例展示（真实项目案例）

你的素材：
> "重构了3个项目，平均每个省了5小时。"

AI会改写成：
> "最近用Claude Code重构了几个老项目。数据说话：平均每个项目节省5小时，主要省在重复代码处理上。"

## 重要原则

### ✅ 必须遵守

1. **所有素材必须是真实的**
   - 不能编造经历
   - 不能夸大数据
   - 不能捏造观点

2. **AI会改写，不是直接复制**
   - 改写成长文逻辑
   - 调整语气和节奏
   - 融入上下文

3. **保护隐私**
   - 不要放涉及他人隐私的内容
   - 敏感信息要脱敏

### ❌ 禁止做的

1. **不能编造素材投喂给AI**
   - AI会相信你的素材是真实的
   - 如果你编造，最终文章会失真

2. **不能直接复制粘贴**
   - AI会改写，不是原文照搬
   - 保持真实性的同时优化表达

## 如何开始？

### 最简单的方法（推荐新手）

1. 创建一个 `我的真实经历.md` 文件
2. 想想最近做过什么、有什么想法
3. 用最自然的语言写下来（就像发朋友圈一样）
4. 不用多，10-20条就够了

**示例：**
```markdown
# 我的真实经历

## AI工具
- 用Claude Code写了个Todo应用，20分钟搞定，之前要2小时
- Cursor的补全功能很准，但有时候会过度补全
- ChatGPT帮我调试了一个bug，节省了1小时

## 产品开发
- 上周做需求调研，发现用户最关心的不是功能，是稳定性
- 重构代码时发现，80%的bug来自同一个模块

## 生活
- 高德扫街榜为了冲榜走了5公里，腿疼...
```

### 进阶方法（如果你有历史数据）

1. 导出你的社交媒体数据（即刻、Twitter等）
2. 保存为CSV或文本文件
3. 放入这个文件夹
4. AI会自动搜索关键词，找到相关素材

## 维护建议

### 定期更新

- 每次有新的想法、经历，随手记录
- 不用追求完美，真实最重要
- 建议每周花10分钟整理一次

### 质量 > 数量

- 10条高质量真实经历 > 100条空洞记录
- 重点记录：具体数字、具体场景、真实感受

### 分类整理（可选）

如果素材多了，可以按主题分类：
- `AI工具经历.md`
- `产品开发心得.md`
- `创业踩坑.md`

## 常见问题

### Q1: 我没有那么多素材怎么办？

**A:** 不用多！10-20条真实经历就够了。关键是真实、具体。

### Q2: 素材太私人了，能不能不放？

**A:** 可以脱敏。比如把具体公司名改成"某公司"，把具体人名改成"同事"。

### Q3: AI会不会直接复制我的素材？

**A:** 不会。AI会改写成长文逻辑，调整语气，融入上下文。你的素材是原料，AI会加工。

### Q4: 我可以编一些素材吗？

**A:** 强烈不建议！AI会相信你的素材是真实的。如果你编造，最终文章会失真，读者会看出来。
