# 视频创作 - 流程

视频创作需要考虑画面、声音、节奏，比文章创作更复杂。

## 视频的特点

- **时间限制**：需要精确控制时长
- **视听结合**：文案 + 画面 + 声音
- **节奏感**：快慢结合，保持吸引力
- **平台差异**：抖音、视频号、B站等平台特点不同

---

## 7步创作流程

### Step 1: 明确需求

**关键问题：**
- 视频平台？（抖音/视频号/B站/YouTube等）
- 视频时长？（15秒/1分钟/3分钟/10分钟等）
- 视频类型？（知识分享/产品介绍/Vlog/教程等）
- 目标受众？（创业者/管理者/普通用户等）
- 拍摄方式？（真人出镜/PPT讲解/动画/混剪等）

---

### Step 2: 信息搜索与验证（如需）

**什么时候需要搜索：**
- 涉及新产品、新技术
- 需要准确数据
- 需要业界最新动态

保存到：`../_knowledge_base/主题-YYYY-MM-DD.md`

---

### Step 3: 选题和结构讨论 ⭐

**不要直接写脚本！先讨论选题和结构。**

**提供：**
- 2-3个视频选题方向
- 每个选题的核心角度
- 大致结构（开头、主体、结尾）
- 预计时长分配

**等待用户选择**

---

### Step 4: 使用个人素材库 ⭐

从 `../个人素材库/` 搜索相关经历、观点、案例。

视频需要更多真实感：
- 真实经历作为案例
- 真实数据支撑观点
- 真实感受引发共鸣

---

### Step 5: 创作脚本

**脚本包含：**

#### 5.1 基本信息
```
标题：[视频标题]
时长：[目标时长，如3分钟]
平台：[抖音/视频号/B站等]
类型：[知识分享/产品介绍/教程等]
```

#### 5.2 完整脚本

**格式：**
```
【画面】          【文案/台词】                    【时长】

开头
---------------------------------------------------------------
[画面描述]        [说什么]                          0:00-0:10

主体
---------------------------------------------------------------
[画面描述]        [说什么]                          0:10-0:30
[画面描述]        [说什么]                          0:30-1:00
...

结尾
---------------------------------------------------------------
[画面描述]        [说什么]                          2:50-3:00

---------------------------------------------------------------
总时长：3分钟
```

**示例：**
```
标题：3分钟学会降低AI写作的AI味
时长：3分钟
平台：视频号
类型：知识分享

【画面】          【文案/台词】                    【时长】

开头
---------------------------------------------------------------
真人出镜          你有没有发现，AI写的文章           0:00-0:05
特写              一看就是AI写的？

画面切换          今天教你一个方法，                 0:05-0:10
文字动画          让AI写的文章更像人写的。

主体
---------------------------------------------------------------
屏幕录制          核心方法就是：                     0:10-0:20
标注重点          用真实经历替代AI腔。

分屏展示          左边是AI直接写的，                 0:20-0:40
对比效果          右边是用真实经历改写的。
                  你看，差别是不是很明显？

真人出镜          具体怎么做？三个步骤...            0:40-2:50
画面演示          [详细演示]

结尾
---------------------------------------------------------------
真人出镜          好了，今天的分享就到这里。         2:50-2:58
字幕              觉得有用的话，点个赞👍

品牌露出          我是[你的名字]，下期见！           2:58-3:00

---------------------------------------------------------------
总时长：3分钟
字数：约450字（按每分钟150字计算）
```

#### 5.3 文案要求

**开头（前5秒最关键）：**
- 提出痛点/问题（引发共鸣）
- 或直接抛出价值（吸引注意）
- 不要铺垫太长

**主体：**
- 结构清晰（第一、第二、第三）
- 节奏紧凑（不拖沓）
- 有具体案例/数据/对比
- 适当重复核心观点（加深印象）

**结尾：**
- 总结核心观点
- 行动号召（点赞、关注、评论等）
- 品牌露出

**语言风格：**
- 口语化（像说话一样）
- 短句为主（3-10字一句）
- 避免长句（超过20字要拆分）
- 适当停顿（用标点体现）

---

### Step 6: 审校优化

**重点检查：**

**6.1 时长控制**
- [ ] 是否符合目标时长？
- [ ] 每个部分时长分配合理？
- [ ] 按每分钟150-180字计算（口语速度）

**6.2 节奏感**
- [ ] 开头是否吸引人？（前5秒）
- [ ] 主体是否紧凑？（不拖沓）
- [ ] 是否有节奏变化？（快慢结合）

**6.3 口语化**
- [ ] 删除书面词汇
- [ ] 句子够短吗？
- [ ] 读起来自然吗？（大声朗读测试）

**6.4 画面配合**
- [ ] 文案和画面是否匹配？
- [ ] 重点是否有视觉强调？
- [ ] 是否考虑了画面转场？

---

### Step 7: 补充说明

**提供：**

**7.1 拍摄要点**
- 需要拍摄哪些画面
- 拍摄角度和景别建议
- 注意事项

**7.2 后期要点**
- 剪辑节奏建议
- 字幕要求
- 配乐建议
- 特效使用建议

**7.3 发布建议**
- 标题建议（2-3个备选）
- 封面建议
- 话题标签建议
- 发布时间建议

## 不同平台的特点

### 抖音/视频号（短视频）

**时长：** 15秒-3分钟（1分钟左右最佳）

**特点：**
- 快节奏，前3秒最关键
- 垂直屏幕（9:16）
- 强调视觉冲击
- 配乐很重要

**脚本要求：**
- 开门见山，不铺垫
- 高密度信息（每5秒一个点）
- 适当留悬念（引导看完）

### B站/YouTube（中长视频）

**时长：** 3-30分钟

**特点：**
- 深度内容
- 横屏（16:9）
- 可以有铺垫和详细展开
- 注重逻辑和完整性

**脚本要求：**
- 可以有简短铺垫（但不超过30秒）
- 结构分明（章节化）
- 深入讲解
- 结尾可以延伸话题

## 常见视频类型模板

### 类型1: 知识分享

**结构：**
```
开头（10%）：提出问题/痛点
主体（80%）：解决方案（分步骤讲解）
结尾（10%）：总结 + 行动号召
```

**时长分配（以3分钟为例）：**
- 开头：20秒
- 主体：2分20秒
- 结尾：20秒

### 类型2: 产品介绍

**结构：**
```
开头（10%）：痛点/使用场景
主体（80%）：产品功能演示
结尾（10%）：购买/咨询引导
```

### 类型3: Vlog/故事分享

**结构：**
```
开头（10%）：抛出话题/故事线索
主体（80%）：故事展开（冲突、转折、高潮）
结尾（10%）：感悟/升华
```

---

## 核心原则

1. **开头最重要**
   - 前3-5秒决定用户是否继续看
   - 直接抛出价值或制造悬念

2. **节奏要紧凑**
   - 删除所有废话
   - 每句话都有价值
   - 快慢结合，避免单调

3. **视听结合**
   - 文案和画面要配合
   - 重点处要有视觉强调
   - 配乐要匹配情绪

4. **真实感**
   - 用真实经历和案例
   - 不要读稿子感
   - 像和朋友聊天
