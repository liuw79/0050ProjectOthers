# 朋友圈写作 - 流程

朋友圈写作相比公众号更简短、更轻松、更即时。

## 朋友圈的特点

- **字数限制**：一般不超过300字
- **风格轻松**：更像日常聊天
- **必须配图**：1-9张图
- **即时性强**：通常快速创作发布

---

## 5步快速流程

### Step 1: 理解需求

**目标：** 明确朋友圈的核心内容和目的。

**关键问题：**
- 想表达什么？（分享、感悟、推广、通知等）
- 目标读者是谁？（朋友、客户、潜在用户等）
- 需要几张图？
- 语气如何？（轻松、正式、感性、理性等）

---

### Step 2: 搜索验证（如需）

**什么时候需要搜索：**
- 涉及新产品、新技术
- 需要准确数据
- 需要业界最新动态

**如果是：**
- 个人感悟、日常分享 → 跳过，直接创作
- 产品推广、行业观点 → 搜索验证

---

### Step 3: 使用个人素材库 ⭐

**这是朋友圈的核心！**

朋友圈最忌讳"假大空"，必须用真实经历和感受。

**使用方法：**
- 从 `../个人素材库/` 搜索相关经历
- 提取真实的感受、想法、吐槽
- 改写成朋友圈风格（更轻松、更口语化）

---

### Step 4: 创作文案

**朋友圈文案的特点：**
- 开门见山（前3行最重要，超过3行会被折叠）
- 口语化（像说话一样）
- 有画面感（具体场景、具体感受）
- 真实感（不装、不端着）

**结构建议：**

**类型A：个人分享/感悟**
```
开头：直接说事（今天...、刚刚...、最近...）
↓
展开：具体描述（场景、感受、思考）
↓
结尾：一句话总结或留白
```

**类型B：产品推广/干货分享**
```
开头：痛点/现象（你是不是也...）
↓
解决方案：介绍产品/方法
↓
结尾：行动号召（扫码、咨询、点赞等）
```

**类型C：通知/公告**
```
核心信息放前3行
↓
详细说明
↓
联系方式/操作指引
```

**文案要求：**
- 字数：50-300字（超过300字会显得太长）
- 段落：1-2行为一段（手机阅读友好）
- 标点：多用句号、问号，少用逗号连接长句
- Emoji：适量使用（1-3个），不要过度

---

### Step 5: 配图建议 ⭐

**配图是朋友圈的灵魂！**

**配图数量：**
- 1张：适合产品图、海报、重要通知
- 3张：平衡、好看
- 4张：方正、规整
- 9张：丰富、完整故事

**配图类型：**
- 产品推广：产品图、使用场景、效果对比
- 个人分享：生活照、风景、美食
- 干货分享：截图、图表、思维导图
- 活动通知：海报、二维码、现场照

**配图要求：**
- 清晰（至少800x800px）
- 统一风格（色调、滤镜）
- 注意隐私（不泄露敏感信息）

**配图保存位置：**
`../images/朋友圈/YYYY-MM-DD-主题/`

---

## 审校要点（降AI味）

朋友圈的审校相对简单，重点关注：

**删除这些表达：**
- ❌ "在当今时代"
- ❌ "综上所述"
- ❌ "值得注意的是"
- ❌ "深深地"、"深刻地"
- ❌ 所有书面词汇

**改成口语化：**
- ❌ "显著提升" → ✅ "好了很多"
- ❌ "充分利用" → ✅ "用好"
- ❌ "进行" → ✅ 直接用动词

**加入真实感：**
- 具体数字（"3天"而不是"几天"）
- 具体场景（"昨天下午"而不是"最近"）
- 真实感受（"累死了"而不是"比较疲惫"）

---

## 朋友圈的核心原则

1. **真实 > 完美**
   - 真实的经历 > 完美的表述
   - 真实的感受 > 漂亮的金句

2. **简短 > 详细**
   - 能3句话说清楚，不说5句
   - 前3行最重要

3. **场景 > 道理**
   - 具体场景 > 抽象道理
   - "我做了什么" > "应该怎么做"

4. **口语 > 书面**
   - 像说话一样写
   - 读起来自然

## 特殊场景

### 场景1: 批量创作朋友圈（一周计划）

如果需要规划一周的朋友圈内容：

1. 明确本周主题（产品推广、个人品牌、日常分享等）
2. 列出7条朋友圈的主题和角度
3. 逐条创作
4. 标注建议发布时间

### 场景2: 系列朋友圈（连续剧式）

如果需要多条朋友圈讲一个完整故事：

1. 规划整体结构（开头、发展、高潮、结尾）
2. 每条朋友圈独立成篇，但有连贯性
3. 前后呼应（比如"接上条"）

## 常见类型模板

### 类型1: 个人感悟

```
[具体事件]

[感受/思考]

[一句话总结或留白]
```

**示例：**
```
今天连续开了4个会，晚上10点才回家。

发现一个规律：真正重要的决策，
往往不是在会议室做出的，
而是在会议结束后，几个人
边走边聊时冒出来的。

有时候，形式越轻松，思考越深入。
```

### 类型2: 产品推广

```
[痛点描述]

[产品/解决方案介绍]

[效果/数据]

[行动号召]
```

**示例：**
```
写文章最烦什么？AI味太重！

我搭了个内容写作系统，
用真实经历替代AI腔。
测试下来，AI检测率从70%降到30%。

有兴趣的朋友可以私聊我👇
```

### 类型3: 干货分享

```
[吸引人的开头]

[核心内容/方法]

[补充说明]
```

**示例：**
```
分享一个降低AI味的方法：

不要让AI直接写文章，
而是先让它搜索你的真实经历，
再基于这些经历创作。

真实经历是降AI味的核心。
```

## 注意事项

1. **避免硬广**
   - 不要每条都是产品推广
   - 个人分享和产品推广比例建议 7:3

2. **注意频率**
   - 一天1-2条为宜
   - 避免刷屏

3. **尊重隐私**
   - 涉及他人时征得同意
   - 敏感信息打码

4. **适度互动**
   - 可以在文案中提问，引发互动
   - 但不要强求点赞、评论
